## BABOK 6 Knowledge Areas

1. **Business Analysis Planning and Monitoring** - Plan approach and governance
2. **Elicitation and Collaboration** - Gather information from stakeholders
3. **Requirements Life Cycle Management** - Trace, prioritize, manage
4. **Strategy Analysis** - Define business need, justify investment
5. **Requirements Analysis and Design Definition** - Specify requirements
6. **Solution Evaluation** - Assess performance, identify improvements

## Core Capabilities

- **Stakeholder Analysis**: Power/Interest grid, RACI
- **Process Mapping**: BPMN notation, value stream mapping
- **Requirements Elicitation**: Interviews, workshops, document analysis
- **Financial Analysis**: ROI, NPV, IRR, Payback Period
- **Gap Analysis**: Current state → Future state → Gaps
- **Solution Evaluation**: Weighted scoring, decision matrices

## Output Artifacts

- Business Requirements Document (BRD)
- Business Case with Financial Analysis
- Stakeholder Register and Analysis
- Process Maps (BPMN/Mermaid)
- Gap Analysis Report
- Requirements Traceability Matrix

## Key Principles

- **Stakeholder-Centric**: Understand all perspectives
- **Evidence-Based**: Support conclusions with data
- **Traceable**: Link business needs → requirements → solutions
- **Pragmatic**: Focus on value delivery
