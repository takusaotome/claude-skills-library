# Corrective Action Tracker

## Overview

This template provides a structured format for tracking corrective actions identified during root cause analysis. It includes 3D classification, SMART criteria validation, ownership assignment, and progress tracking. Use this template during Workflow 7 (Corrective Action Planning).

## Review Meeting Template

### Agenda for Corrective Action Review Meeting

1. **Dashboard Review** (5 min)
   - Overall completion rate
   - Overdue actions
   - Blocked actions

2. **Action-by-Action Status** (20 min)
   - Review each in-progress action
   - Review each blocked action and identify resolution
   - Review each overdue action and agree on new timeline or escalation

3. **New Risks or Dependencies** (5 min)
   - Any new information affecting actions?
   - Any new risks identified since last review?

4. **Next Steps and Assignments** (5 min)
   - Confirm next review date
   - Confirm ownership for any reassigned actions

## Closure Criteria

The corrective action tracker can be closed when ALL of the following are satisfied:

- [ ] All immediate actions are completed and verified
- [ ] All short-term actions are completed and verified
- [ ] All long-term actions are completed and verified OR formally deferred with leadership approval
- [ ] No actions remain in "Blocked" status
- [ ] 3D coverage is at least 2/3 for each root cause
- [ ] Effectiveness of completed actions has been validated (e.g., test alert fires correctly)
- [ ] Final summary report has been distributed to stakeholders
- [ ] Tracker owner has signed off on closure
