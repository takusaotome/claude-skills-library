# 戦略分析フレームワーク詳細ガイド

このドキュメントは、外部環境分析と内部環境分析のための主要フレームワークの詳細な実施手法を提供します。

## 1. PEST/PESTLE分析

### 概要

PEST分析は、マクロ環境要因を体系的に分析するフレームワークです。PESTLEはPESTを拡張し、Legal（法律）とEnvironmental（環境）を追加したものです。

### 分析目的

- 事業環境の変化を早期に察知する
- 機会と脅威を特定する
- 戦略立案の前提条件を明確化する

### 各要素の詳細分析ポイント

#### P - Political（政治的要因）

**分析対象:**
- 政府の安定性と政策方向性
- 規制環境の変化（規制強化/緩和）
- 税制変更（法人税、消費税、関税）
- 貿易政策と国際関係
- 産業政策と補助金制度
- 政治的リスク（政権交代、選挙）

**分析質問例:**
1. 今後3-5年で予想される規制変更は何か？
2. 政府の産業政策は自社にどう影響するか？
3. 国際関係の変化がサプライチェーンに与える影響は？
4. 税制改正の事業への影響は？

**評価指標:**
| 指標 | 測定方法 |
|------|---------|
| 規制リスク | 規制変更の頻度と影響度 |
| 政策支援度 | 補助金・優遇措置の有無 |
| 政治安定性 | 政権交代リスク評価 |

#### E - Economic（経済的要因）

**分析対象:**
- GDP成長率と景気動向
- インフレ率と金利動向
- 為替レートの変動
- 失業率と賃金水準
- 消費者購買力と消費動向
- 資本市場の状況（株価、資金調達環境）

**分析質問例:**
1. 景気循環は現在どのフェーズにあるか？
2. 金利変動が設備投資計画にどう影響するか？
3. 為替変動が輸出入コストに与える影響は？
4. 消費者の購買力変化が需要にどう影響するか？

**主要経済指標:**
| 指標 | 影響 | 確認頻度 |
|------|------|---------|
| GDP成長率 | 全体需要 | 四半期 |
| CPI（消費者物価指数） | 価格転嫁力 | 月次 |
| 政策金利 | 資金調達コスト | 随時 |
| 為替レート | 輸出入採算 | 日次 |

#### S - Social（社会的要因）

**分析対象:**
- 人口動態（少子高齢化、人口移動）
- ライフスタイルの変化
- 消費者の価値観・嗜好の変化
- 教育水準と労働力の質
- 健康・環境意識の高まり
- ワークスタイルの変化

**分析質問例:**
1. 主要顧客層の人口動態変化は？
2. 消費者の価値観変化がニーズにどう影響するか？
3. 労働力確保の観点で社会変化の影響は？
4. 新しい消費トレンドへの対応は必要か？

**トレンド分析:**
| トレンド | 影響領域 | 対応方向性 |
|---------|---------|-----------|
| 高齢化 | 製品設計、サービス | バリアフリー、シニア向け |
| 健康志向 | 製品開発 | 健康価値の訴求 |
| SDGs意識 | ブランド、調達 | サステナビリティ対応 |

#### T - Technological（技術的要因）

**分析対象:**
- 技術革新のスピードと方向性
- デジタル化・DXの進展
- 新技術の商業化タイミング
- 研究開発動向（産学連携、国家プロジェクト）
- 技術標準・規格の動向
- 技術の陳腐化リスク

**分析質問例:**
1. 破壊的技術が事業モデルに与える影響は？
2. デジタル化対応の遅れによるリスクは？
3. 技術標準の変更が製品に与える影響は？
4. 競合の技術開発動向は？

**技術成熟度評価:**
| 技術 | 成熟度 | 自社対応状況 | 優先度 |
|------|--------|-------------|--------|
| AI/ML | 普及期 | 実証段階 | 高 |
| IoT | 成長期 | 導入済み | 中 |
| ブロックチェーン | 黎明期 | 未対応 | 低 |

#### L - Legal（法律的要因）

**分析対象:**
- 労働法規（働き方改革、最低賃金）
- 知的財産保護
- 消費者保護法規
- データ保護・プライバシー規制
- 業界固有の法規制
- 訴訟リスク

**コンプライアンス評価:**
| 法規制 | 対応状況 | リスク | 対策 |
|--------|---------|--------|------|
| 個人情報保護法 | 対応済 | 低 | 定期監査 |
| 下請法 | 一部対応 | 中 | 契約見直し |
| 製造物責任法 | 対応済 | 中 | 保険加入 |

#### E - Environmental（環境的要因）

**分析対象:**
- 環境規制の強化
- 気候変動への対応要請
- 資源・エネルギー制約
- 廃棄物・リサイクル規制
- ESG投資の拡大
- カーボンニュートラル目標

**環境対応評価:**
| 項目 | 現状 | 目標 | ギャップ |
|------|------|------|---------|
| CO2排出量 | 1,000t | 500t | 500t削減必要 |
| 再エネ比率 | 30% | 100% | 70%向上必要 |
| リサイクル率 | 85% | 95% | 10%向上必要 |

## 2. Porter's Five Forces（ポーターの5つの力）

### 概要

マイケル・ポーターが提唱した業界構造分析フレームワーク。業界の収益性を決定する5つの競争要因を分析します。

### 5つの競争要因

#### 1. 業界内の競争（Industry Rivalry）

**競争激化の要因:**
- 競合企業数が多い
- 業界成長率が低い
- 固定費が高い（稼働率競争）
- 製品差別化が困難
- スイッチングコストが低い
- 撤退障壁が高い
- 戦略的利害が大きい

**評価チェックリスト:**
| 評価項目 | スコア(1-5) | 判断基準 |
|---------|------------|---------|
| 競合数 | | 多い(5)〜少ない(1) |
| 市場成長率 | | 低い(5)〜高い(1) |
| 製品差別化 | | 困難(5)〜容易(1) |
| 固定費比率 | | 高い(5)〜低い(1) |
| 撤退障壁 | | 高い(5)〜低い(1) |

#### 2. 新規参入の脅威（Threat of New Entrants）

**参入障壁の要因:**
- 規模の経済性
- 製品差別化（ブランド力）
- 必要資本量
- スイッチングコスト
- 流通チャネルへのアクセス
- 規制・許認可要件
- 既存企業の報復予想

**参入障壁評価:**
| 障壁要因 | 高さ | 説明 |
|---------|------|------|
| 規模の経済 | 高/中/低 | 最小効率規模は大きいか |
| 資本要件 | 高/中/低 | 初期投資はいくら必要か |
| ブランド力 | 高/中/低 | 既存ブランドの優位性 |
| 規制障壁 | 高/中/低 | 許認可取得の難易度 |

#### 3. 代替品の脅威（Threat of Substitutes）

**代替品の脅威が高まる条件:**
- 代替品の相対価格が低い
- 代替品の品質・性能が同等以上
- スイッチングコストが低い
- 顧客の代替品志向が高い

**代替品分析:**
| 代替品 | 価格比 | 機能比 | スイッチングコスト | 脅威度 |
|--------|--------|--------|------------------|--------|
| 製品A | 80% | 90% | 低 | 高 |
| 製品B | 120% | 110% | 中 | 中 |
| サービスC | 60% | 70% | 高 | 中 |

#### 4. 買い手の交渉力（Bargaining Power of Buyers）

**買い手の交渉力が高まる条件:**
- 買い手の集中度が高い
- 購入量が大きい
- 製品が標準化されている
- スイッチングコストが低い
- 後方統合の脅威がある
- 製品が買い手のコストに占める割合が大きい
- 買い手の情報が豊富

**買い手分析:**
| 顧客セグメント | 売上比率 | 交渉力 | 対策 |
|--------------|---------|--------|------|
| 大手小売 | 40% | 高 | 差別化製品提案 |
| 中堅卸売 | 30% | 中 | 関係構築 |
| 小規模店舗 | 30% | 低 | 標準サービス |

#### 5. 売り手の交渉力（Bargaining Power of Suppliers）

**売り手の交渉力が高まる条件:**
- 供給者の集中度が高い
- 代替供給源が少ない
- 製品が差別化されている
- スイッチングコストが高い
- 前方統合の脅威がある
- 買い手の事業に対する重要性が高い

**サプライヤー分析:**
| サプライヤー | 調達比率 | 交渉力 | リスク軽減策 |
|------------|---------|--------|-------------|
| 原材料A社 | 50% | 高 | 代替供給源開拓 |
| 部品B社 | 30% | 中 | 長期契約 |
| 設備C社 | 20% | 低 | 複数社並行取引 |

### 戦略的示唆の導出

| 競争要因 | 評価 | 戦略的対応 |
|---------|------|-----------|
| 業界内競争 | 高 | 差別化戦略、ニッチ集中 |
| 新規参入 | 中 | 参入障壁構築、ブランド強化 |
| 代替品 | 低 | 価値訴求、ロックイン強化 |
| 買い手の力 | 高 | 顧客分散、価値提案強化 |
| 売り手の力 | 中 | 調達先多様化、垂直統合検討 |

## 3. Value Chain分析（バリューチェーン分析）

### 概要

マイケル・ポーターが提唱した、企業の価値創造活動を分析するフレームワーク。主活動と支援活動に分類し、競争優位の源泉を特定します。

### バリューチェーン構造

```

│ 経営企画、財務、法務、品質管理                                      │
├────────────────────────────────────────────────────────────────────┤
│ 人事・労務管理                                                      │
│ 採用、教育訓練、報酬制度、労働環境                                  │
├────────────────────────────────────────────────────────────────────┤
│ 技術開発                                                            │
│ R&D、製品開発、プロセス改善、IT                                     │
├────────────────────────────────────────────────────────────────────┤
│ 調達活動                                                            │
```

### 主活動の詳細分析

#### 購買物流（Inbound Logistics）

**分析ポイント:**
- 原材料・部品の受け入れ効率
- 在庫管理の最適化
- 品質検査の精度
- サプライヤーとの情報連携

**評価項目:**
| 項目 | 現状評価 | ベンチマーク | ギャップ |
|------|---------|-------------|---------|
| 在庫回転率 | 8回/年 | 12回/年 | -4回 |
| 受入リードタイム | 3日 | 1日 | +2日 |
| 不良受入率 | 2% | 0.5% | +1.5% |

#### 製造・オペレーション（Operations）

**分析ポイント:**
- 生産効率（OEE: 総合設備効率）
- 品質レベル（不良率、歩留まり）
- 柔軟性（多品種少量対応）
- コスト競争力

**評価項目:**
| 項目 | 現状評価 | ベンチマーク | ギャップ |
|------|---------|-------------|---------|
| OEE | 75% | 85% | -10% |
| 不良率 | 1.5% | 0.5% | +1.0% |
| 段取り時間 | 30分 | 10分 | +20分 |

#### 出荷物流（Outbound Logistics）

**分析ポイント:**
- 配送リードタイム
- 配送コスト効率
- 配送品質（破損率、誤配率）
- 在庫配置の最適化

#### 販売・マーケティング（Marketing & Sales）

**分析ポイント:**
- ブランド認知度・評価
- 販売チャネルの効率性
- 価格競争力
- 顧客獲得コスト

#### サービス（Service）

**分析ポイント:**
- 顧客満足度
- サポート対応スピード
- 保守・メンテナンス品質
- アップセル・クロスセル

### 支援活動の詳細分析

#### 全般管理（Firm Infrastructure）

- 経営計画・管理プロセスの有効性
- 財務管理能力
- 法務・コンプライアンス体制
- 情報システム基盤

#### 人事・労務管理（Human Resource Management）

- 人材採用力
- 人材育成・能力開発
- 従業員エンゲージメント
- 報酬・評価制度の競争力

#### 技術開発（Technology Development）

- R&D投資効率
- イノベーション創出力
- 知的財産ポートフォリオ
- 技術の事業化スピード

#### 調達活動（Procurement）

- 調達コスト競争力
- サプライヤー関係管理
- 調達リスク管理
- 戦略的調達能力

### 競争優位の源泉特定

バリューチェーン分析から、以下を特定する：

1. **コスト優位の源泉**
   - どの活動でコスト削減が可能か
   - 規模の経済が効く活動はどれか
   - 効率改善の余地がある活動は

2. **差別化優位の源泉**
   - 顧客価値に最も貢献する活動は
   - 競合と差別化できる活動は
   - 模倣困難な強みはどこにあるか

3. **フォーカス戦略の可能性**
   - 特定セグメントに集中すべき活動は
   - 外部委託すべき活動は
   - 重点投資すべき活動は

## 4. 分析フレームワークの統合

### 分析品質チェックリスト

- [ ] PEST: 6要素すべてを網羅したか
- [ ] PEST: 各要素に3つ以上のトレンドを特定したか
- [ ] PEST: 機会/脅威の判定を行ったか
- [ ] 5F: 5つの力すべてを評価したか
- [ ] 5F: 定量的なデータで裏付けたか
- [ ] 5F: 業界魅力度を総合評価したか
- [ ] Value Chain: 9つの活動すべてを評価したか
- [ ] Value Chain: 競合比較を行ったか
