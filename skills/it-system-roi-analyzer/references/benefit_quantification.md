# IT効果定量化ガイド

## Overview

ITシステム導入による効果を定量的に金額換算するための方法論。経営層への説得力あるビジネスケース作成のため、可能な限り効果を数値化する。

---

## 効果の分類

### 効果カテゴリ

```
IT導入効果
├── 1. コスト削減効果（Hard Benefits）
│   ├── 人件費削減
│   ├── IT運用コスト削減
│   ├── 外注費削減
│   └── その他経費削減
├── 2. 生産性向上効果（Hard Benefits）
│   ├── 処理時間短縮
│   ├── スループット向上
│   ├── エラー削減
│   └── 意思決定迅速化
├── 3. リスク低減効果（Semi-Hard Benefits）
│   ├── セキュリティ強化
│   ├── 可用性向上
│   ├── コンプライアンス強化
│   └── 事業継続性向上
└── 4. 売上・収益貢献効果（Hard Benefits）
    ├── 新規顧客獲得
    ├── 顧客単価向上
    ├── 解約率低減
    └── 新規事業創出
```

---

## 1. コスト削減効果

### 人件費削減

**計算式:**
```
人件費削減 = 削減工数（時間/月） × 時間単価 × 12ヶ月
```

**例:**
| 項目 | 削減工数 | 時間単価 | 年間削減額 |
|------|---------|---------|-----------|
| データ入力自動化 | 100時間/月 | 3,000円 | 360万円 |
| レポート作成自動化 | 40時間/月 | 4,000円 | 192万円 |
| 承認フロー電子化 | 20時間/月 | 5,000円 | 120万円 |

**時間単価の算出:**
```
時間単価 = 年間人件費 ÷ 年間労働時間
         = (月給 × 12 + 賞与 + 福利厚生費) ÷ (8時間 × 20日 × 12ヶ月)
         ≒ 年収 × 1.3～1.5 ÷ 1,920時間
```

**目安時間単価（日本）:**
| 職種 | 時間単価 |
|------|---------|
| 一般事務 | 2,500-3,500円 |
| 専門職 | 4,000-6,000円 |
| 管理職 | 6,000-10,000円 |
| 役員 | 10,000円以上 |

---

### IT運用コスト削減

**計算式:**
```
IT運用コスト削減 = 現行運用コスト - 新システム運用コスト
```

**削減項目例:**
| 項目 | 現行 | 新システム | 削減額 |
|------|------|-----------|--------|
| サーバー費用 | 1,000万円/年 | 500万円/年 | 500万円 |
| ライセンス費 | 300万円/年 | 100万円/年 | 200万円 |
| 保守費用 | 500万円/年 | 300万円/年 | 200万円 |
| データセンター費 | 200万円/年 | 0円（クラウド移行） | 200万円 |

---

### 外注費削減

**計算式:**
```
外注費削減 = 削減する外注業務 × 外注単価
```

**例:**
- データ入力外注: 500万円/年 → 自動化により0円 = 500万円削減
- 帳票印刷外注: 100万円/年 → 電子化により0円 = 100万円削減

---

## 2. 生産性向上効果

### 処理時間短縮

**計算式:**
```
処理時間短縮効果 = 短縮時間 × 時間単価 × 処理回数
```

**例:**
| 業務 | 現行時間 | 新時間 | 短縮 | 回数/年 | 単価 | 効果額 |
|------|---------|--------|------|--------|------|--------|
| 受注処理 | 30分 | 5分 | 25分 | 12,000回 | 3,000円/時 | 1,500万円 |
| 見積作成 | 2時間 | 30分 | 1.5時間 | 1,000回 | 4,000円/時 | 600万円 |
| 月次決算 | 5日 | 2日 | 3日 | 12回 | 50万円/日 | 1,800万円 |

---

### スループット向上

**計算式:**
```
スループット向上効果 = 追加処理能力 × 単位当たり利益
```

**例:**
- 製造ライン効率化: 処理能力20%向上 × 年間売上10億円 × 利益率10% = 2,000万円
- コールセンター: 対応件数30%増 × 追加売上3,000万円 × 利益率15% = 450万円

---

### エラー削減

**計算式:**
```
エラー削減効果 = 削減エラー件数 × エラー1件当たり対応コスト
```

**エラー対応コストの構成:**
- 発見・調査時間
- 修正作業時間
- 再テスト時間
- 顧客対応時間
- 機会損失

**例:**
| エラー種別 | 現行件数 | 新件数 | 削減 | 対応コスト | 効果額 |
|-----------|---------|--------|------|-----------|--------|
| データ入力ミス | 50件/月 | 5件/月 | 45件 | 5万円/件 | 2,700万円/年 |
| 請求ミス | 10件/月 | 1件/月 | 9件 | 10万円/件 | 1,080万円/年 |
| 在庫差異 | 100件/月 | 20件/月 | 80件 | 2万円/件 | 1,920万円/年 |

## 3. リスク低減効果

### セキュリティリスク低減

**計算式:**
```
セキュリティ効果 = 想定損失額 × 発生確率低減
```

**情報漏洩の想定損失:**
| 規模 | 想定損失 |
|------|---------|
| 小規模（1,000件未満） | 1,000-5,000万円 |
| 中規模（1万件未満） | 5,000万-2億円 |
| 大規模（10万件以上） | 2-10億円以上 |

**例:**
- 個人情報漏洩リスク: 想定損失2億円 × 発生確率5%低減 = 1,000万円/年相当
- ランサムウェア被害: 想定損失5,000万円 × 発生確率3%低減 = 150万円/年相当

---

### 可用性向上（ダウンタイム削減）

**計算式:**
```
可用性向上効果 = 削減ダウンタイム × 時間当たり損失
```

**時間当たり損失の算出:**
| 業種 | 1時間当たり損失（目安） |
|------|----------------------|
| EC/小売 | 売上÷営業時間 |
| 製造業 | 生産高÷稼働時間 |
| 金融 | 取引高×手数料率÷営業時間 |
| サービス業 | 売上÷営業時間 |

**例:**
- EC サイト: ダウンタイム年10時間→1時間削減、時間損失200万円 = 1,800万円/年

---

### コンプライアンス強化

**計算式:**
```
コンプライアンス効果 = 罰金・訴訟リスク × 発生確率低減
```

**主な罰則・リスク:**
| 法規制 | 最大罰則 |
|--------|---------|
| 個人情報保護法 | 1億円 |
| GDPR | 2,000万ユーロ or 売上4% |
| PCI DSS違反 | 取引停止、賠償 |
| 内部統制不備 | 上場廃止リスク |

---

## 4. 売上・収益貢献効果

### 新規顧客獲得

**計算式:**
```
新規顧客効果 = 増加顧客数 × 顧客生涯価値（LTV）
```

**LTVの算出:**
```
LTV = 平均購入額 × 購入頻度 × 継続期間 × 利益率
```

**例:**
- CRM導入で月10顧客増 × LTV 50万円 = 年間6,000万円

---

### 顧客単価向上

**計算式:**
```
顧客単価向上効果 = 単価増加額 × 購入回数
```

**例:**
- レコメンドエンジン導入: 平均購入額1,000円増 × 月1万件 = 年間1.2億円

---

### 解約率低減（チャーン低減）

**計算式:**
```
解約率低減効果 = 保持顧客数 × 顧客年間売上
```

**例:**
- 顧客サポート強化: 解約率15%→10%（5%低減）× 顧客売上1億円 = 年間500万円

## 定性的効果の扱い

### 定量化が難しい効果

| 効果 | 間接的な定量化アプローチ |
|------|------------------------|
| 従業員満足度向上 | 離職率低減→採用・育成コスト削減 |
| ブランドイメージ向上 | 広告宣伝費換算 |
| 意思決定スピード向上 | 機会損失削減（推定） |
| 競争優位性確保 | 市場シェア維持→売上維持 |

### 記載方法

**財務指標計算には含めず、追加価値として別記:**

```
## 定量的効果（財務指標算出対象）
- コスト削減: XX万円/年
- 生産性向上: XX万円/年
- 合計: XX万円/年

## 追加的価値（定量化困難）
- 従業員満足度向上（eNPS改善見込み）
- ブランドイメージ向上（顧客アンケート改善見込み）
- 意思決定スピード向上（経営会議効率化見込み）
- 将来の事業拡張性確保
```

## 効果発現のタイミング

### 段階的効果発現モデル

| 期間 | 効果発現率 | 理由 |
|------|-----------|------|
| Year 1 | 50% | 導入・習熟期間 |
| Year 2 | 100% | 安定稼働 |
| Year 3 | 100% | 最適化 |
| Year 4 | 100% | 成熟期 |
| Year 5 | 100% | 成熟期 |

**キャッシュフロー計算への反映:**
```
Year 1効果 = 年間効果額 × 50%
Year 2以降効果 = 年間効果額 × 100%
```

## 効果測定KPI

### 効果検証のためのKPI設定

| 効果カテゴリ | KPI例 |
|-------------|-------|
| コスト削減 | IT運用費（月次）、人件費（月次） |
| 生産性向上 | 処理時間（業務別）、エラー率 |
| リスク低減 | インシデント件数、ダウンタイム |
| 売上貢献 | 新規顧客数、解約率、平均購入額 |

### 効果測定の頻度

- **月次**: 運用コスト、処理時間、エラー率
- **四半期**: 売上指標、顧客満足度
- **年次**: ROI達成度、TCO実績
