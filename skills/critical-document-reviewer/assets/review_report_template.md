### B. 用語定義

| 用語 | 定義 |
|------|------|
| Critical | 根拠なし、または重大な論理誤謬。このまま進めると失敗リスク大 |
| Major | 根拠が不十分、または実害を招く可能性のある曖昧さ |
| Minor | 改善が望ましいが、致命的ではない |
| Info | 参考情報、改善提案 |

### C. 根拠評価レベル

| Level | 評価 | 定義 |
|-------|------|------|
| 5 | Verified | 客観的データ・証拠で検証済み |
| 4 | Reliable | 信頼できる情報源からの引用 |
| 3 | Reasonable | 合理的な推論で導出 |
| 2 | Insufficient | 根拠はあるが不完全 |
| 1 | None | 根拠が示されていない |
