# シナジー分析ガイド

## 概要

M&Aにおけるシナジー（相乗効果）の特定、定量化、実現計画策定のためのガイド。シナジーはM&A価値創造の源泉であり、適切な分析と実現計画が成功の鍵となる。

---

## 1. シナジーの類型

### 1.1 シナジー分類

| 類型 | 内容 | 実現難易度 | 実現期間 |
|-----|-----|----------|---------|
| **コストシナジー** | 費用削減による効果 | 中 | 1-3年 |
| **収益シナジー** | 売上増加による効果 | 高 | 2-5年 |
| **財務シナジー** | 資金調達・税務効果 | 低 | 即時-1年 |

### 1.2 各シナジー類型の詳細

#### コストシナジー

| カテゴリ | 具体例 |
|---------|-------|
| 本社機能統合 | 経理、人事、法務、IT部門の統合 |
| 調達コスト削減 | 購買統合によるスケールメリット |
| 拠点統合 | オフィス、工場、倉庫の統廃合 |
| システム統合 | IT基盤の共通化 |
| 人員削減 | 重複ポジションの解消 |
| マーケティング効率化 | 広告宣伝費の統合 |
| 製造効率化 | 生産ラインの最適化 |

#### 収益シナジー

| カテゴリ | 具体例 |
|---------|-------|
| クロスセル | 相手方顧客への製品販売 |
| アップセル | 顧客単価向上 |
| 販売チャネル活用 | 販路の相互利用 |
| 地理的拡大 | 新市場への参入 |
| 製品開発加速 | R&D資源の統合 |
| ブランド活用 | ブランド価値の相互活用 |
| 価格交渉力 | 市場シェア拡大による価格力 |

#### 財務シナジー

| カテゴリ | 具体例 |
|---------|-------|
| 資金調達コスト低減 | 信用力向上による金利低下 |
| 税務効果 | 繰越欠損金の活用 |
| 運転資本効率化 | キャッシュマネジメントの改善 |
| 内部資金活用 | 余剰資金の有効活用 |

## 3. コストシナジーの定量化

### 3.1 本社機能統合

**算定方法:**
```
シナジー = 統合後削減人員 × 人件費単価
        + 拠点統合によるコスト削減
        - 退職金・再配置コスト
```

**ベンチマーク:**
| 機能 | 削減率目安 |
|-----|----------|
| 経理・財務 | 15-30% |
| 人事・総務 | 15-25% |
| IT | 20-35% |
| 法務・コンプライアンス | 10-20% |
| 経営企画 | 20-30% |

### 3.2 調達コスト削減

**算定方法:**
```
シナジー = 統合調達額 × 削減率
```

**削減率目安:**
| 調達カテゴリ | 削減率目安 |
|------------|----------|
| 直接材（原材料） | 3-8% |
| 間接材（消耗品等） | 5-15% |
| 外注・委託費 | 5-10% |
| IT関連 | 10-20% |
| 物流費 | 5-12% |

### 3.3 拠点統合

**算定方法:**
```
シナジー = 閉鎖拠点の賃料・維持費
        - 移転・統合コスト（一時費用）
        - 残存賃料・違約金
```

### 3.4 人員削減

**算定方法:**
```
シナジー = 削減人員数 × 年間人件費
実現コスト = 削減人員数 × 退職パッケージ（通常1-2年分）
```

---

## 4. 収益シナジーの定量化

### 4.1 クロスセル・アップセル

**算定方法:**
```
シナジー = 対象顧客数 × クロスセル率 × 製品単価 × 粗利率
```

**成功率の目安:**
| シナリオ | クロスセル成功率 |
|---------|----------------|
| 楽観的 | 15-25% |
| 現実的 | 5-15% |
| 保守的 | 2-8% |

### 4.2 新市場・チャネル

**算定方法:**
```
シナジー = 新チャネル経由売上増 × 粗利率
        = 既存製品 × 新チャネル浸透率 × 単価
```

### 4.3 価格交渉力

**算定方法:**
```
シナジー = 統合後売上高 × 価格引上げ率 × 粗利への貢献率
```

## 5. 実現確度評価

### 5.1 確度評価マトリックス

| 確度レベル | 基準 | NPV反映率 |
|----------|-----|----------|
| 確実（90%+） | 契約済み、実績あり | 100% |
| 高（70-90%） | 具体計画策定済み | 80% |
| 中（50-70%） | 実行可能性確認済み | 60% |
| 低（30-50%） | コンセプト段階 | 40% |
| 要検証（<30%） | 仮説段階 | 0%（計上せず） |

### 5.2 確度評価の視点

| 評価項目 | 確認内容 |
|---------|---------|
| 実行可能性 | 技術的・組織的に実行可能か |
| 定量化の根拠 | 算定ロジックの信頼性 |
| 実現の前例 | 類似取引での実績 |
| 実行体制 | 責任者、リソースの確保 |
| 外部依存 | 顧客・取引先の協力要否 |
| リスク要因 | 阻害要因の有無 |

### 5.3 シナジー類型別の実現確度傾向

| シナジー類型 | 一般的な確度 | 備考 |
|------------|------------|-----|
| 本社統合 | 高 | 自社でコントロール可能 |
| 調達統合 | 高～中 | サプライヤー交渉次第 |
| 人員削減 | 高～中 | 労務リスクあり |
| システム統合 | 中 | 技術的複雑性 |
| クロスセル | 中～低 | 顧客反応次第 |
| 新製品開発 | 低 | 不確実性高 |

## 7. シナジー価値のNPV算定

### 7.1 シナジーNPV計算

```
シナジーNPV = Σ (シナジー効果t × 確度) / (1+r)^t - 実現コスト

r: 割引率（通常WACCを使用）
t: 年数
```

### 7.2 算定例

```
シナジー項目: 調達コスト削減
- 年間効果額: 10億円
- 実現確度: 80%
- 実現コスト: 2億円（Year 1）
- 実現スケジュール: Y1 30%, Y2 70%, Y3+ 100%
- 割引率: 8%

Year 1: 10億円 × 30% × 80% / 1.08 = 2.2億円
Year 2: 10億円 × 70% × 80% / 1.08^2 = 4.8億円
Year 3+: 10億円 × 100% × 80% / (0.08 × 1.08^2) = 85.7億円（永続価値）

シナジーNPV = 2.2 + 4.8 + 85.7 - 2 = 90.7億円
```

## 8. リスク要因と対策

### 8.1 シナジー実現のリスク

| リスク要因 | 内容 | 対策 |
|----------|-----|-----|
| 統合の遅延 | 計画通りに進まない | 詳細計画、マイルストーン管理 |
| 人材流出 | キーパーソンの退職 | リテンションパッケージ |
| 顧客離反 | 統合に伴う顧客離脱 | 早期コミュニケーション |
| 文化衝突 | 組織文化の不一致 | 統合後の文化醸成 |
| 技術的障壁 | システム統合の複雑さ | 段階的統合 |
| 規制対応 | 独禁法、業法対応 | 早期当局確認 |

### 8.2 シナジーディスシナジー（マイナス効果）

| ディスシナジー | 内容 |
|--------------|-----|
| 統合コスト | 一時費用の発生 |
| 生産性低下 | 統合期間中の混乱 |
| 顧客離反 | ブランド変更等による離脱 |
| 人材流出コスト | 代替採用・育成コスト |
| システム問題 | 統合トラブルによる損失 |
