# 財務デューデリジェンス（Financial Due Diligence）チェックリスト

## 概要

財務DDは、対象企業の財務状態、収益力、キャッシュフロー創出能力を検証し、買収価格の妥当性と潜在的リスクを評価する。

---

## 1. 財務諸表分析

### 1.1 損益計算書（P/L）分析

| チェック項目 | 確認内容 | リスクレベル |
|------------|---------|------------|
| □ 売上高推移 | 過去3-5年の成長率、季節性、顧客集中度 | |
| □ 売上認識基準 | 収益認識タイミング、基準変更の有無 | |
| □ 売上総利益率 | 製品/サービス別利益率、推移分析 | |
| □ 販管費構造 | 固定費/変動費分解、人件費比率 | |
| □ 営業利益率 | 業界平均との比較、改善/悪化要因 | |
| □ EBITDA調整 | 一時費用、非経常項目の調整 | |
| □ 純利益率 | 税効果、特別損益の影響 | |

**主要確認事項:**
- 正常収益力（Normalized Earnings）の算定
- EBITDA調整項目の妥当性検証
- 顧客上位10社への売上集中度（30%超は要注意）

### 1.2 貸借対照表（B/S）分析

| チェック項目 | 確認内容 | リスクレベル |
|------------|---------|------------|
| □ 現預金 | 運転資金水準、遊休資金の有無 | |
| □ 売掛金 | 回収期間、滞留債権、貸倒引当金 | |
| □ 棚卸資産 | 回転期間、陳腐化リスク、評価方法 | |
| □ 固定資産 | 減価償却方法、減損リスク、遊休資産 | |
| □ 無形資産 | のれん、ソフトウェア、償却計画 | |
| □ 買掛金 | 支払条件、仕入先集中度 | |
| □ 借入金 | 金利条件、返済スケジュール、コベナンツ | |
| □ 純資産 | 資本構成、配当政策、自己資本比率 | |

**主要確認事項:**
- 運転資本（Working Capital）の正常水準
- ネットデット（Net Debt）の算定
- 簿外債務・偶発債務の有無

### 1.3 キャッシュフロー計算書（C/F）分析

| チェック項目 | 確認内容 | リスクレベル |
|------------|---------|------------|
| □ 営業CF | P/Lとの整合性、運転資本変動 | |
| □ 投資CF | 設備投資水準、維持更新投資 | |
| □ 財務CF | 配当、借入返済、資金調達 | |
| □ FCF | フリーキャッシュフロー創出力 | |
| □ 現金変換サイクル | CCC（Cash Conversion Cycle） | |
