# DCF法（Discounted Cash Flow）バリュエーション手法ガイド

## 概要

DCF法は、将来のフリーキャッシュフロー（FCF）を適切な割引率で現在価値に割り引くことで企業価値を算定する手法。インカムアプローチの代表的手法。

---

## 1. DCF法の基本フレームワーク

### 1.1 企業価値算定式

```
企業価値（EV） = Σ FCFt / (1+WACC)^t + TV / (1+WACC)^n

株式価値 = 企業価値 - ネットデット - 非支配株主持分 + 非事業用資産

ネットデット = 有利子負債 - 現預金
```

### 1.2 DCFプロセス

```
[Step 1: 事業計画検証] → [Step 2: FCF予測] → [Step 3: WACC算定]
        ↓                       ↓                    ↓
[Step 4: 継続価値算定] → [Step 5: 現在価値算定] → [Step 6: 感度分析]
```

---

## 2. フリーキャッシュフロー（FCF）予測

### 2.1 FCF計算式

**アンレバードFCF（Unlevered FCF）:**
```
FCF = NOPAT + 減価償却費 - 設備投資 - 運転資本増減

NOPAT = EBIT × (1 - 税率)
```

| 項目 | 計算 |
|-----|-----|
| 売上高 | |
| (-) 売上原価 | |
| (-) 販管費 | |
| = EBIT（営業利益） | |
| (-) 税金（EBIT × 税率） | |
| = NOPAT | |
| (+) 減価償却費 | |
| (-) 設備投資 | |
| (-) 運転資本増減 | |
| = **FCF** | |

### 2.2 予測期間

| 項目 | 標準 | 考慮事項 |
|-----|-----|---------|
| 予測期間 | 5-10年 | 事業の成熟度、予測の信頼性 |
| 明示的予測期間 | 5年 | 詳細な事業計画がある期間 |
| フェードイン期間 | 5年 | 成長率が永久成長率に収束 |

### 2.3 事業計画の検証

| 確認項目 | チェックポイント |
|---------|-----------------|
| 過去実績との整合性 | 成長率、利益率の連続性 |
| 市場成長率との整合性 | 市場シェアの変化を暗示 |
| 設備投資と売上の整合性 | 生産能力との対応 |
| 運転資本の推移 | 売上成長との連動 |
| シナジー効果の織り込み | 独立評価かシナジー込みか |

## 3. WACC（加重平均資本コスト）算定

### 3.1 WACC計算式

```
WACC = E/(D+E) × Re + D/(D+E) × Rd × (1-T)

E: 株式の時価
D: 有利子負債の時価
Re: 株主資本コスト
Rd: 負債コスト
T: 実効税率
```

### 3.2 株主資本コスト（Re）- CAPM

```
Re = Rf + β × (Rm - Rf) + サイズプレミアム + 国別リスクプレミアム

Rf: リスクフリーレート
β: ベータ（市場感応度）
Rm - Rf: 株式リスクプレミアム（ERP）
```

### 3.3 各パラメータの算定

#### リスクフリーレート（Rf）

| 市場 | 指標 | 参照 |
|-----|-----|-----|
| 日本 | 10年国債利回り | 財務省 |
| 米国 | 10年Treasury | Fed |
| ユーロ | 10年Bund | ECB |

#### β（ベータ）

```
アンレバードβ = レバードβ / [1 + (1-T) × D/E]
レバードβ = アンレバードβ × [1 + (1-T) × D/E]
```

| 算出方法 | 内容 |
|---------|-----|
| 類似企業βの平均 | 上場類似企業のβを平均 |
| 業界βの参照 | 業界平均値を使用 |
| 回帰分析 | 対象会社（上場の場合） |

#### 株式リスクプレミアム（ERP）

| 市場 | 目安 | ソース |
|-----|-----|-------|
| 日本 | 5.0-7.0% | Ibbotson, Duff & Phelps |
| 米国 | 5.5-6.5% | Duff & Phelps |
| 新興国 | +1.0-4.0% | 国別調整 |

#### サイズプレミアム

| 時価総額 | サイズプレミアム | 参考 |
|---------|-----------------|-----|
| 大型（1兆円超） | 0.0% | |
| 中型（1000億-1兆円） | 1.0-2.0% | |
| 小型（100億-1000億円） | 2.0-4.0% | |
| 極小（100億円未満） | 4.0-6.0% | |

### 3.4 負債コスト（Rd）

| 算出方法 | 内容 |
|---------|-----|
| 実績借入金利 | 有利子負債の加重平均金利 |
| 信用格付対応利回り | 格付に応じた社債利回り |

### 3.5 資本構成

| アプローチ | 内容 |
|----------|-----|
| 現状資本構成 | 現在のD/E比率を使用 |
| 目標資本構成 | 経営計画上の目標D/E |
| 類似企業平均 | 業界平均D/E |
| 最適資本構成 | 理論的最適値 |

### 3.6 WACC算定例

```
前提:
- Rf = 0.8%（日本10年国債）
- アンレバードβ = 0.9（類似企業平均）
- ERP = 6.0%
- サイズプレミアム = 2.0%
- Rd = 1.5%
- T = 30%
- D/E = 30%

計算:
レバードβ = 0.9 × [1 + (1-0.3) × 0.3] = 1.09
Re = 0.8% + 1.09 × 6.0% + 2.0% = 9.3%
E/(D+E) = 1 / 1.3 = 76.9%
D/(D+E) = 0.3 / 1.3 = 23.1%

WACC = 76.9% × 9.3% + 23.1% × 1.5% × (1-0.3)
     = 7.2% + 0.2% = 7.4%
```

## 4. 継続価値（Terminal Value）算定

### 4.1 永久成長率モデル（Gordon Growth Model）

```
TV = FCFn+1 / (WACC - g)
   = FCFn × (1+g) / (WACC - g)

g: 永久成長率
n: 予測期間の最終年
```

#### 永久成長率（g）の設定

| 考慮要因 | 目安 |
|---------|-----|
| 長期インフレ率 | 1.0-2.0%（日本） |
| GDP成長率 | 0.5-1.5%（日本） |
| 業界成長率 | 業界により異なる |
| 一般的レンジ | 0.0-3.0% |

**注意**: g > WACCにならないよう注意

### 4.2 Exit Multipleモデル

```
TV = EBITDAn × Exit Multiple
```

#### Exit Multipleの設定

| 設定方法 | 内容 |
|---------|-----|
| 類似企業マルチプル | 現在の類似企業EV/EBITDA |
| 取引マルチプル | 過去取引のマルチプル |
| Entry Multiple | 買収時マルチプルを使用 |

### 4.3 継続価値の現在価値

```
TV現在価値 = TV / (1+WACC)^n
```

**確認ポイント**: TV現在価値が企業価値全体の60-80%程度が一般的
