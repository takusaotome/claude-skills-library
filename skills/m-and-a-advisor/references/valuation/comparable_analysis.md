# 類似企業比較法（Comparable Companies Analysis）バリュエーション手法ガイド

## 概要

類似企業比較法は、類似する上場企業のバリュエーションマルチプルを参照して対象企業の価値を算定するマーケットアプローチの手法。

---

## 1. 類似企業比較法の基本

### 1.1 基本算定式

```
対象企業の企業価値（EV）= 対象企業の財務指標 × 類似企業のマルチプル

例: EV = 対象企業EBITDA × 類似企業EV/EBITDA倍率
```

### 1.2 主要マルチプル

| マルチプル | 計算式 | 適用場面 |
|----------|-------|---------|
| EV/EBITDA | 企業価値 / EBITDA | 最も一般的、資本構成の影響排除 |
| EV/EBIT | 企業価値 / 営業利益 | 減価償却が重要な業種 |
| EV/Sales | 企業価値 / 売上高 | 赤字企業、成長企業 |
| P/E | 株価 / EPS | 成熟企業、安定収益企業 |
| PBR | 株価 / BPS | 金融機関、資産集約型 |
| EV/Subscribers | 企業価値 / 加入者数 | 通信、SaaS等 |

## 2. 類似企業の選定

### 2.1 選定基準

| 基準 | 確認項目 |
|-----|---------|
| 業種・事業内容 | 同一または類似の事業セグメント |
| ビジネスモデル | B2B/B2C、ストック型/フロー型 |
| 事業規模 | 売上高、時価総額が近い |
| 成長性 | 売上成長率、利益成長率 |
| 収益性 | 営業利益率、EBITDA マージン |
| 地理的展開 | 国内/グローバル |
| 事業段階 | 成長期/成熟期 |

### 2.2 類似企業の特定方法

1. **業界分類から特定**
   - 東証業種分類
   - GICS（Global Industry Classification Standard）
   - 証券アナリスト業種区分

2. **競合企業から特定**
   - 対象企業の有価証券報告書の競合記載
   - 業界レポート、アナリストレポート

3. **財務プロファイルでスクリーニング**
   - 売上規模
   - 成長率
   - 利益率

### 2.3 選定企業数の目安

| 状況 | 企業数 |
|-----|-------|
| 理想的 | 5-10社 |
| 最小 | 3社 |
| 類似企業が少ない場合 | 3-5社 + 調整コメント |

---

## 3. マルチプル算定

### 3.1 EV（企業価値）の計算

```
EV = 株式時価総額 + ネットデット + 非支配株主持分 + 優先株式

ネットデット = 有利子負債 - 現預金（および現金同等物）
```

### 3.2 財務指標の選択

| 指標 | 期間 | 考慮事項 |
|-----|-----|---------|
| LTM（直近12ヶ月） | 過去実績 | 最新の実績を反映 |
| NTM（今後12ヶ月） | 予想 | 将来の期待を反映 |
| 当期予想 | FY予想 | アナリスト予想を使用 |
| 来期予想 | FY+1予想 | 成長企業向け |

### 3.3 EBITDA調整

| 調整項目 | 内容 |
|---------|-----|
| 一時費用 | リストラ費用、訴訟費用等を加算 |
| 一時収益 | 資産売却益等を減算 |
| ストックオプション費用 | 非現金費用として加算調整 |
| リース費用 | IFRS16/ASC842対応 |

---

## 4. プレミアム/ディスカウント調整

### 4.1 コントロールプレミアム

| 状況 | プレミアム目安 |
|-----|--------------|
| 支配権取得（100%） | 20-40% |
| 過半数取得（50%超） | 15-30% |
| 少数持分取得 | なし |

**適用条件**: 類似企業マルチプルは少数株主ベースのため、支配権取得時に加算

### 4.2 流動性ディスカウント（非上場企業の場合）

| 状況 | ディスカウント目安 |
|-----|-----------------|
| 売却容易（IPO候補等） | 10-20% |
| 一般的な非上場企業 | 20-30% |
| 売却困難（事業継承等） | 25-35% |

### 4.3 その他調整要因

| 調整要因 | 方向 | 調整幅目安 |
|---------|-----|----------|
| 成長率が高い | プレミアム | +10-30% |
| 成長率が低い | ディスカウント | -10-20% |
| 収益性が高い | プレミアム | +5-15% |
| 収益性が低い | ディスカウント | -5-15% |
| 規模が小さい | ディスカウント | -10-20% |
| ブランド力 | プレミアム | +5-15% |

## 5. 業種別マルチプル目安

### 5.1 日本市場ベンチマーク（参考値）

| 業種 | EV/EBITDA | EV/Sales | P/E |
|-----|----------|---------|-----|
| 製造業（一般） | 6-10x | 0.8-1.5x | 12-18x |
| 自動車 | 4-7x | 0.3-0.7x | 8-14x |
| IT・ソフトウェア | 10-20x | 2-6x | 20-40x |
| SaaS | 15-30x | 5-15x | 30-60x |
| 通信 | 5-8x | 1.5-3x | 10-15x |
| 小売（一般） | 5-8x | 0.3-0.8x | 12-20x |
| EC | 10-25x | 1-4x | 25-50x |
| 金融（銀行） | N/A | N/A | 8-12x |
| 不動産 | 10-15x | 2-5x | 12-18x |
| ヘルスケア | 10-18x | 2-5x | 20-35x |
| 食品・飲料 | 8-12x | 1-2x | 15-25x |

**注意**: 市況により大きく変動するため、評価時点での最新データを使用すること

### 5.2 成長ステージ別調整

| ステージ | マルチプル傾向 |
|---------|--------------|
| 成長期（高成長） | 高マルチプル（売上成長重視） |
| 安定期（成熟） | 中マルチプル（利益重視） |
| 衰退期 | 低マルチプル（資産価値重視） |

## 6. 分析手順

### Step 1: 類似企業リストの作成

```
| 企業名 | ティッカー | 時価総額 | 売上高 | EBITDA | 営業利益 |
|-------|----------|---------|-------|--------|---------|
| A社   |          |         |       |        |         |
| B社   |          |         |       |        |         |
```

### Step 2: マルチプルの算定

```
| 企業名 | EV | EV/EBITDA | EV/Sales | P/E |
|-------|-----|----------|---------|-----|
| A社   |     |          |         |     |
| B社   |     |          |         |     |
| 平均   |     |          |         |     |
| 中央値 |     |          |         |     |
```

### Step 3: 外れ値の除外

- 平均から±2標準偏差以上は除外検討
- 特殊要因（M&A直後、業績悪化等）がある企業は除外

### Step 4: 対象企業への適用

```
対象企業EV = 対象企業EBITDA × 選定マルチプル
対象企業株式価値 = EV - ネットデット ± 調整項目
```

### Step 5: レンジの提示

```
| マルチプル | 下限（25%ile） | 中央値 | 上限（75%ile） |
|----------|--------------|--------|--------------|
| EV/EBITDA |              |        |              |
| 株式価値   |              |        |              |
```

## 7. 注意事項

### 7.1 マルチプル適用時の注意

- **会計基準の違い**: IFRS/日本基準/US-GAAPの違いを調整
- **決算期の違い**: 比較可能な期間に調整
- **一時的要因**: 非経常項目を調整したNormalized指標を使用
- **資本構成の違い**: レバレッジ差異の影響を考慮

### 7.2 限界

- 類似企業が存在しない場合は適用困難
- 市場全体の過熱/冷え込みの影響を受ける
- 対象企業固有の成長性・リスクを反映しにくい
- 非上場企業には直接適用できない（調整が必要）

### 7.3 他手法との併用

- DCF法と併用し、クロスチェックを行う
- 先行取引分析法とも比較検証
- 複数手法の結果をフットボールチャートで提示
