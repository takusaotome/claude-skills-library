# 先行取引分析法（Precedent Transactions Analysis）バリュエーション手法ガイド

## 概要

先行取引分析法は、類似するM&A取引で実際に支払われた価格（マルチプル）を参照して対象企業の価値を算定するマーケットアプローチの手法。

---

## 1. 先行取引分析法の基本

### 1.1 基本概念

```
対象企業の企業価値 = 対象企業の財務指標 × 先行取引のマルチプル

特徴: 実際の取引価格にはコントロールプレミアムとシナジー価値が含まれる
```

### 1.2 類似企業比較法との違い

| 項目 | 先行取引分析法 | 類似企業比較法 |
|-----|--------------|--------------|
| 参照データ | 過去のM&A取引価格 | 上場企業の株価 |
| コントロールプレミアム | 含まれる | 含まれない |
| シナジー価値 | 一部含まれる可能性 | 含まれない |
| データの入手性 | 限定的（非公開多い） | 容易 |
| 時点 | 取引時点（過去） | 現時点 |

---

## 2. 類似取引の選定

### 2.1 選定基準

| 基準 | 確認項目 | 重要度 |
|-----|---------|-------|
| 業種・事業内容 | 同一または類似の事業 | 最重要 |
| 取引規模 | 取引価額が近い | 高 |
| 取引時期 | 直近3-5年以内 | 高 |
| 地域 | 同一または類似市場 | 中 |
| 取引形態 | 100%取得、過半数取得等 | 中 |
| 買い手属性 | 戦略的買収/ファイナンシャル | 中 |

### 2.2 取引選定のフィルター

**第1次スクリーニング:**
- 業種コード（GICS、SIC等）
- 取引規模（取引価額下限）
- 取引時期（過去5年以内）
- 地域（ターゲット所在地）

**第2次スクリーニング:**
- ビジネスモデルの類似性
- 成長性・収益性の比較
- 取引の背景・動機

### 2.3 取引情報のソース

| ソース | 内容 | 備考 |
|-------|-----|-----|
| M&Aデータベース | Refinitiv, Bloomberg, Mergermarket | 有料 |
| 適時開示 | TDnet、EDINET | 無料 |
| プレスリリース | 買収発表資料 | 無料 |
| 業界レポート | 投資銀行、コンサル資料 | 有料/限定 |
| ニュース記事 | M&A報道 | 無料 |

## 3. 取引マルチプル分析

### 3.1 主要マルチプル

| マルチプル | 計算式 | 特徴 |
|----------|-------|-----|
| EV/EBITDA | 取引価額 / LTM EBITDA | 最も一般的 |
| EV/EBIT | 取引価額 / LTM EBIT | 減価償却の影響排除 |
| EV/Sales | 取引価額 / LTM 売上高 | 赤字企業にも適用可 |
| P/E | 株式取得価額 / LTM EPS | 少数持分取得時 |

### 3.2 取引データ収集項目

```
| 取引 | 発表日 | ターゲット | 買い手 | 取引価額 | EV | 売上高 | EBITDA | EV/EBITDA | EV/Sales |
|-----|-------|----------|-------|---------|-----|-------|--------|----------|---------|
| #1  |       |          |       |         |     |       |        |          |         |
| #2  |       |          |       |         |     |       |        |          |         |
```

### 3.3 財務指標の基準時点

| オプション | 内容 | 使用場面 |
|----------|-----|---------|
| LTM（取引発表時） | 発表直前の12ヶ月実績 | 最も一般的 |
| NTM（取引発表時） | 発表時点の将来12ヶ月予想 | 成長企業 |
| 当期予想 | 取引発表時点の通期予想 | 予想が公表されている場合 |

## 4. コントロールプレミアム分析

### 4.1 コントロールプレミアムの定義

```
コントロールプレミアム = (取引価格 - 市場株価) / 市場株価 × 100%

市場株価: 通常、取引発表1ヶ月前の終値または1ヶ月平均株価
```

### 4.2 プレミアム水準の目安

| 市場・時期 | 平均プレミアム | 中央値 |
|----------|--------------|-------|
| 日本市場（全体） | 30-40% | 25-35% |
| 米国市場（全体） | 30-45% | 25-40% |
| 敵対的買収 | 40-60% | 35-50% |
| MBO | 30-50% | 30-40% |

### 4.3 プレミアムに影響する要因

| 要因 | プレミアム傾向 |
|-----|--------------|
| 競合入札 | 高い |
| 敵対的買収 | 高い |
| シナジー大 | 高い |
| 売り手の交渉力 | 高い |
| 市場環境（好況） | 高い |
| 戦略的買い手 | 高い（vs ファイナンシャル） |

## 5. シナジー価値の扱い

### 5.1 取引価格に含まれるシナジー

**注意**: 先行取引のマルチプルには、買い手が見込んだシナジー価値が一部反映されている可能性がある

### 5.2 シナジーの調整方法

| アプローチ | 内容 |
|----------|-----|
| 調整なし | シナジー込みのマルチプルをそのまま使用 |
| 開示シナジー控除 | 公表されたシナジー見込額を控除 |
| 推定シナジー控除 | 業界平均シナジー率で推定・控除 |

### 5.3 シナジー調整の例

```
公表取引価額: 1,000億円
公表シナジー見込: 100億円（シナジーマルチプル10x想定）
スタンドアローン価値推定: 1,000 - 100 = 900億円

シナジー調整後マルチプル = 900 / EBITDA
```

## 6. 分析手順

### Step 1: 取引リストの作成

1. M&Aデータベースで類似取引を検索
2. 選定基準に基づきフィルタリング
3. 取引詳細情報を収集

### Step 2: 取引データの整理

```
| # | 発表日 | ターゲット | 業種 | 買い手 | 取引形態 | 取引価額 |
|---|-------|----------|-----|-------|---------|---------|
| 1 |       |          |     |       | 100%買収 |         |
| 2 |       |          |     |       | TOB     |         |
```

### Step 3: マルチプル算定

```
| # | EV | LTM売上 | LTM EBITDA | EV/Sales | EV/EBITDA |
|---|-----|--------|-----------|---------|----------|
| 1 |     |        |           |         |          |
| 2 |     |        |           |         |          |
| 平均 |   |        |           |         |          |
| 中央値 | |        |           |         |          |
```

### Step 4: 外れ値・異常値の検討

- 特殊な取引背景（救済合併、敵対的買収等）
- 異常に高い/低いマルチプル
- データの信頼性

### Step 5: 対象企業への適用

```
適用マルチプル（中央値使用推奨）
対象企業EV = 対象企業EBITDA × 先行取引EV/EBITDA中央値
```

### Step 6: 調整・検証

- 市場環境の変化を考慮（古い取引は調整）
- コントロールプレミアムの妥当性確認
- DCF・類似企業比較法との整合性確認

## 8. 注意事項・限界

### 8.1 データの制約

- 非上場企業間取引は情報が限定的
- 取引価額が非公開の場合が多い
- 財務データの詳細が不明な場合がある

### 8.2 比較可能性の限界

- 取引ごとに条件・背景が異なる
- シナジー期待値が取引ごとに異なる
- 市場環境（金利、株価水準）が異なる

### 8.3 時間的制約

- 古い取引は市場環境が異なる
- 直近取引が少ない場合、サンプルが限定的

### 8.4 活用上のポイント

- 複数手法（DCF、類似企業比較）と併用
- 取引背景・条件を十分に理解した上で使用
- マルチプルのレンジで結論を提示
- 調整・コメントを適切に付記
