## 1. ユニットエコノミクスの構築

収益予測の基本は「1単位あたりの経済性」を明確にすること。

### SaaS / サブスクリプションモデル

**主要指標:**
- **MRR (Monthly Recurring Revenue)**: 月次定期収入
- **ARPU (Average Revenue Per User)**: ユーザーあたり平均収入
- **CAC (Customer Acquisition Cost)**: 顧客獲得コスト
- **LTV (Lifetime Value)**: 顧客生涯価値
- **Churn Rate**: 月次解約率
- **LTV/CAC比率**: 3以上が健全（投資回収の目安）

**計算式:**
```
LTV = ARPU × 粗利率 / 月次チャーン率
CAC = (マーケティング費 + 営業費) / 新規獲得顧客数
Payback Period = CAC / (ARPU × 粗利率)
```

**月次推移モデル:**
```
月初顧客数 = 前月末顧客数
新規顧客 = マーケティング投資 / CAC
解約顧客 = 月初顧客数 × チャーン率
月末顧客数 = 月初顧客数 + 新規顧客 - 解約顧客
MRR = 月末顧客数 × ARPU
```

### ECサイト / 物販モデル

**主要指標:**
- **GMV (Gross Merchandise Value)**: 流通総額
- **Take Rate**: 手数料率
- **AOV (Average Order Value)**: 平均注文額
- **購入頻度**: 顧客あたり年間購入回数
- **粗利率**: (売上 - 仕入原価) / 売上

**計算式:**
```
売上 = 訪問者数 × コンバージョン率 × AOV
粗利 = 売上 × 粗利率
顧客あたり年間売上 = AOV × 年間購入頻度
```

### サービス / コンサルティングモデル

**主要指標:**
- **稼働率**: 収益が発生する時間の割合
- **時間単価 / 日単価**: 請求レート
- **プロジェクト平均単価**: 案件あたりの平均売上

**計算式:**
```
売上 = コンサルタント数 × 稼働率 × 営業日数 × 日単価
粗利 = 売上 - 直接人件費
```

### マーケットプレイスモデル

**主要指標:**
- **GMV**: 総取引額
- **テイクレート**: 手数料率
- **リピート率**: 再利用率

**計算式:**
```
プラットフォーム売上 = GMV × テイクレート
GMV = アクティブ出品者数 × 出品者あたり月間取引額
```

## 2. 収益予測の作り方

### ボトムアップ予測（推奨）

個別の営業活動から積み上げる方法。信頼性が高い。

**例: B2B SaaS**
```
月間リード数: 500件（Web広告300 + 紹介100 + コンテンツ100）
商談化率: 20% → 月間商談100件
成約率: 15% → 月間新規15社
ARPU: ¥50,000/月
→ 月間新規MRR: ¥750,000
→ 年間累積（チャーン3%想定）: 約¥7,500,000のMRR
```

### 年次成長の想定

成長率は年度が進むにつれて逓減するのが自然:

| 年度 | 成長率目安（スタートアップ） | 成長率目安（既存事業拡大） |
|---|---|---|
| 1年目 | 100〜300% | 20〜50% |
| 2年目 | 50〜150% | 15〜30% |
| 3年目 | 30〜80% | 10〜20% |
| 4〜5年目 | 20〜50% | 5〜15% |

## 3. 費用計画の作り方

### 費用カテゴリ一覧

**人件費（通常最大の費用項目）:**
- 給与・賞与
- 社会保険料（給与の約15%）
- 福利厚生費
- 採用費（年収の20〜35%がエージェント手数料相場）
- 研修費

**オフィス関連:**
- 家賃・管理費
- 敷金・礼金（初期費用）
- 光熱費
- 通信費（インターネット、電話）
- 備品・消耗品

**IT・システム:**
- サーバー/クラウド費用
- SaaSツール利用料
- 開発外注費
- セキュリティ対策費
- ドメイン・SSL費用

**マーケティング:**
- Web広告費（リスティング、SNS広告）
- コンテンツ制作費
- PR費
- イベント出展費
- ツール利用料（MA、CRM等）

**その他:**
- 法務・会計費用
- 保険料
- 交通費・出張費
- 接待交際費
- 予備費（全体の5〜10%）

### 固変分解

費用を「固定費」と「変動費」に分類し、損益分岐点を算出:

```
損益分岐点売上 = 固定費 / (1 - 変動費率)
損益分岐点顧客数 = 固定費 / (ARPU - 変動費/顧客)
```

## 4. 3シナリオ分析

### シナリオ設定基準

| 項目 | 楽観 | 標準 | 悲観 |
|---|---|---|---|
| 売上成長率 | 計画×1.3 | 計画通り | 計画×0.6 |
| 顧客獲得 | CAC低下、CV率向上 | 計画通り | CAC上昇、CV率低下 |
| チャーン | 想定の50% | 計画通り | 想定の1.5倍 |
| 人件費 | 計画通り | 計画通り | 計画×1.1 |
| 開発遅延 | なし | 1〜2ヶ月 | 3〜6ヶ月 |
| 市場環境 | 追い風 | 現状維持 | 逆風 |

### 各シナリオで確認すべきこと
- **楽観**: 追加投資のタイミングと金額
- **標準**: ブレークイーブンの時期、必要資金額
- **悲観**: キャッシュが尽きるタイミング、撤退基準

## 5. KPI設計

### ステージ別の重要KPI

**シード〜シリーズA:**
- ユーザー数の成長率
- リテンション率
- NPS（顧客満足度）
- バーンレート

**シリーズA〜B:**
- MRR成長率
- LTV/CAC
- 粗利率
- Rule of 40（成長率+利益率≧40%）

**シリーズB以降:**
- Net Revenue Retention
- マジックナンバー
- CAC Payback Period
- フリーキャッシュフロー

### KPI設定のルール
- 3〜5個に絞る（多すぎると集中できない）
- 測定可能であること
- 目標値とタイムラインを明確にする
- 先行指標と遅行指標のバランスを取る

## 6. 財務諸表の作成

### 損益計算書（P/L）テンプレート

```
売上高
 - 売上原価
= 粗利（売上総利益）
 - 販売費及び一般管理費
   - 人件費
   - マーケティング費
   - オフィス費用
   - システム費用
   - その他
= 営業利益
 - 営業外費用（支払利息等）
= 経常利益
 - 税金
= 当期純利益
```

### キャッシュフロー計画

```
月初残高
+ 営業キャッシュフロー（売上入金 - 費用支払）
+ 投資キャッシュフロー（設備投資等）
+ 財務キャッシュフロー（借入、増資等）
= 月末残高
```

**注意:** 売上の計上と入金にはタイムラグがある。
法人取引は通常月末締め翌月末払い（30〜60日のズレ）。

## 7. 資金調達計画

### 必要資金の算出

```
必要資金 = 初期投資 + 運転資金（ブレークイーブンまで）+ 予備資金（3〜6ヶ月分）
```

### 資金調達手段の比較

| 手段 | メリット | デメリット | 適用場面 |
|---|---|---|---|
| 自己資金 | 意思決定自由 | 限度あり | 小規模開始 |
| エンジェル投資 | 経験・人脈 | 株式希薄化 | シード期 |
| VC | 大規模資金 | 株式希薄化、報告義務 | グロース期 |
| 銀行融資 | 株式維持 | 返済義務、担保 | 安定収益あり |
| 補助金・助成金 | 返済不要 | 申請手間、使途制限 | 条件合致時 |
| クラウドファンディング | 事前検証 | 成功不確実 | B2Cプロダクト |

### バリュエーションの目安（シード〜シリーズA）

```
プレマネーバリュエーション = 調達額 / 放出株式比率
（例: ¥100M調達、20%放出 → プレマネー¥400M、ポストマネー¥500M）
```

一般的な放出比率: 15〜25%（1ラウンドあたり）
