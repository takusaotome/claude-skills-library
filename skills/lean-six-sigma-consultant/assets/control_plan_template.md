## Standard Reaction Plan

### Out-of-Control Response Procedure

**Step 1: Immediate Actions**
1. [ ] Mark the out-of-control point on the control chart
2. [ ] Stop production if necessary (based on severity)
3. [ ] Segregate/quarantine affected product
4. [ ] Notify supervisor within [X minutes]

**Step 2: Containment**
1. [ ] Identify potentially affected product (time/lot range)
2. [ ] Quarantine suspect material
3. [ ] Increase inspection frequency

**Step 3: Investigation**
1. [ ] Review recent changes (5M+E)
2. [ ] Check measurement system
3. [ ] Examine materials/inputs
4. [ ] Interview operators

**Step 4: Correction**
1. [ ] Implement corrective action
2. [ ] Verify fix effectiveness
3. [ ] Resume normal production
4. [ ] Document in corrective action system

**Step 5: Follow-up**
1. [ ] Complete investigation report
2. [ ] Update control plan if needed
3. [ ] Communicate lessons learned
