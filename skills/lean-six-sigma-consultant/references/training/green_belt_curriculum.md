# Green Belt Training Curriculum

## Overview

Green Belt certification prepares professionals to lead small to medium Lean Six Sigma projects while maintaining their regular job responsibilities. Green Belts apply statistical tools and Lean principles to solve quality and efficiency problems.

## Target Audience

- Process owners and supervisors
- Quality engineers
- Project leads
- Technical professionals
- Anyone who will lead improvement projects part-time

## Prerequisites

- Yellow Belt certification or equivalent knowledge
- Basic computer skills (Excel)
- Support from management to work on projects

## Training Duration

- 4-6 days (32-48 hours) of classroom training
- Spread over 2-4 months with project work between sessions
- 3-6 months to complete certification project

## Learning Objectives

After completing Green Belt training and project, participants will be able to:
1. Lead DMAIC projects from start to finish
2. Apply appropriate Lean and statistical tools
3. Facilitate project team meetings
4. Present data and findings effectively
5. Implement sustainable improvements
6. Coach Yellow Belts and project team members

## Module 1: Define Phase (4-6 hours)

### Topics

**Business Case Development**
- Problem quantification
- Financial impact
- Strategic alignment

**Project Charter Elements**
- Problem statement (specific, measurable, impact-focused)
- Goal statement (SMART)
- Scope (in/out boundaries)
- Team composition
- Timeline and milestones

**Voice of the Customer**
- Customer identification
- VOC collection methods
  - Surveys
  - Interviews
  - Focus groups
  - Complaint analysis
- CTQ (Critical to Quality) tree
- Translating needs to measurable requirements

**SIPOC Development**
- High-level process view
- Boundary identification
- Key inputs and outputs

**Stakeholder Analysis**
- Identification matrix
- Communication planning
- Change management considerations

### Deliverables
- Completed project charter
- SIPOC diagram
- CTQ tree
- Stakeholder analysis

### Assessment
- Charter presentation to champion/sponsor

---

## Module 2: Measure Phase (8-10 hours)

### Topics

**Data Collection Planning**
- What to measure (Y, Xs)
- Operational definitions
- Sample size considerations
- Collection methods
- Data forms design

**Types of Data**
| Type | Examples | Charts |
|------|----------|--------|
| Continuous | Time, weight, length | Histogram, run chart |
| Discrete - Attribute | Pass/fail, yes/no | Bar chart, Pareto |
| Discrete - Count | Defects, errors | Pareto, C chart |

**Basic Statistics**
- Central tendency (mean, median, mode)
- Variation (range, standard deviation, variance)
- Distribution concepts
- Normal distribution properties

**Process Mapping**
- Detailed flowcharts
- Swim lane diagrams
- Value-add analysis
- Identifying waste

**Measurement System Analysis (MSA)**
- Why MSA matters
- Gage R&R overview
- Attribute Agreement Analysis basics
- Acceptance criteria

**Baseline Metrics**
- Sigma level calculation
- DPMO calculation
- Process capability introduction (Cp, Cpk)
- Yield calculations

**Statistical Software Introduction**
- Minitab or Excel for analysis
- Creating charts and graphs
- Basic statistical outputs

### Deliverables
- Data collection plan
- Process map with value-add analysis
- MSA results
- Baseline metrics
- Initial statistical analysis

### Exercises
- Design data collection for case study
- Conduct mini MSA study
- Calculate baseline sigma level
- Create process map

## Module 3: Analyze Phase (8-10 hours)

### Topics

**Root Cause Analysis**
- Fishbone diagram (detailed)
- 5 Whys technique
- Is/Is Not analysis
- Fault tree analysis

**Graphical Analysis**
- Pareto charts
- Histograms
- Box plots
- Scatter plots
- Run charts

**Statistical Analysis**

*Descriptive Statistics*
- Summary statistics
- Distribution analysis
- Normality testing

*Hypothesis Testing Concepts*
- Null and alternative hypotheses
- Type I and Type II errors
- P-value interpretation
- Confidence intervals

*Comparison Tests*
- Two-sample t-test
- Paired t-test
- One-way ANOVA
- Chi-square test

*Correlation and Regression*
- Correlation coefficient
- Simple linear regression
- Interpreting R-squared

**Verified Root Causes**
- Data validation of suspected causes
- Cause prioritization
- Quantifying impact

### Deliverables
- Root cause analysis documentation
- Statistical analysis summary
- Verified root causes with data support
- Analysis presentation

### Exercises
- Conduct hypothesis tests on sample data
- Analyze case study data set
- Validate suspected root causes

**Sustainability**

- Lessons learned

## Project Requirements

### Certification Project

**Scope**:
- Complete DMAIC project
- Measurable improvement (typically 50%+ improvement in key metric)
- 3-6 month timeline
- Support from champion

**Deliverables**:
1. Completed project storyboard
2. Tollgate presentations (D, M, A, I, C)
3. Financial validation (if applicable)
4. Control plan and handoff documentation
5. Lessons learned

**Common Project Types**:
- Cycle time reduction
- Error/defect reduction
- Cost reduction
- Customer satisfaction improvement
- Productivity improvement

---

## Assessment and Certification

### Requirements

1. **Attend all training sessions** (or make-up)
2. **Pass written examination** (minimum 70%)
3. **Complete certification project** with documented results
4. **Present project** to certification committee
5. **Champion sign-off** on project completion

### Exam Topics

- DMAIC methodology and deliverables
- Statistical concepts and tools
- Lean principles and tools
- Problem-solving techniques
- Project management
- Control methods

### Exam Format

- 80-100 multiple choice questions
- 2-3 hours
- Open book (for some programs)
- Practical application focus

## Post-Certification

### Continuous Development
- Lead additional projects (2-3 per year)
- Mentor Yellow Belts
- Attend advanced training
- Consider Black Belt certification

### Maintaining Skills
- Apply tools regularly
- Join LSS community
- Read case studies
- Teach others

### Career Path
```
Green Belt
    ↓
Lead multiple projects successfully
    ↓
Consider Black Belt training
    ↓
Full-time improvement role (optional)
```
