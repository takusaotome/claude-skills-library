# GRI Standards Quick Reference Guide

## Overview

The GRI (Global Reporting Initiative) Standards are the world's most widely used sustainability reporting standards. This guide provides a quick reference for implementing GRI-aligned reporting.

## GRI Standards Structure (2021 Version)

### Universal Standards (Required for "in accordance" reporting)

| Standard | Title | Purpose |
|----------|-------|---------|
| GRI 1 | Foundation | Reporting principles, requirements |
| GRI 2 | General Disclosures | Organizational profile, governance, strategy |
| GRI 3 | Material Topics | Materiality process, topic management |

### Topic-Specific Standards (Report based on materiality)

#### Environmental (GRI 300 series)
| Standard | Topic | Key Metrics |

| GRI 301 | Materials | Weight/volume, recycled input, reclaimed products |
| GRI 302 | Energy | Consumption within/outside org, intensity, reductions |
| GRI 303 | Water & Effluents | Withdrawal, discharge, consumption by source/destination |
| GRI 304 | Biodiversity | Sites near protected areas, significant impacts |
| GRI 305 | Emissions | Scope 1, 2, 3 GHG emissions, intensity, reductions |
| GRI 306 | Waste | Generation, diversion, disposal methods |
| GRI 308 | Supplier Environmental Assessment | New suppliers screened, impacts identified |

## Reporting Principles

### Principles for Defining Report Content
1. **Stakeholder Inclusiveness**: Identify and respond to stakeholder expectations
2. **Sustainability Context**: Present performance in broader sustainability context
3. **Materiality**: Focus on topics reflecting significant impacts
4. **Completeness**: Sufficient coverage to reflect impacts

### Principles for Defining Report Quality
1. **Accuracy**: Sufficiently detailed and accurate information
2. **Balance**: Report both positive and negative performance
3. **Clarity**: Understandable and accessible
4. **Comparability**: Consistent over time, enable benchmarking
5. **Reliability**: Gathered, recorded, analyzed properly
6. **Timeliness**: Report regularly, timely for stakeholder decisions

## Reporting Levels

### "In Accordance" with GRI Standards
- Apply all GRI Universal Standards
- Report on all material topics using GRI Topic Standards
- Publish GRI Content Index
- Provide statement of use in report

### "With Reference" to GRI Standards
- Selective use of GRI Standards
- Must cite specific GRI disclosures used
- No GRI Content Index required

## Best Practices

1. **Start Early**: Begin data collection in Q1 for previous year report
2. **Automate Data Collection**: Use ESG software platforms where possible
3. **Engage Stakeholders**: Annual materiality review with key stakeholders
4. **Get Assurance**: External assurance increases credibility (limited or reasonable)
5. **Be Transparent**: Report negative performance honestly with corrective actions
6. **Compare to Peers**: Benchmark against industry leaders
7. **Set Targets**: Report progress against quantitative targets
