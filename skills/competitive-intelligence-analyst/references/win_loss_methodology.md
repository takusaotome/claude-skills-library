# Win/Loss Analysis Methodology

商談の勝敗要因を体系的に分析し、営業戦略改善に繋げるための手法ガイド。

---

## Win/Loss分析とは

Win/Loss分析は、商談の勝敗要因を体系的に分析し、営業・マーケティング・製品戦略の改善に活かす手法です。

### 目的と価値

| 目的 | 期待効果 |
|-----|---------|
| 勝因の特定・強化 | 成功パターンの再現性向上 |
| 敗因の特定・改善 | 失注率の低減 |
| 競合戦略の理解 | 効果的な対抗策の立案 |
| 顧客インサイトの獲得 | 製品・サービス改善 |
| 営業プロセスの最適化 | セールスサイクルの短縮 |

---

## 分析プロセス

### Phase 1: 計画策定

**1. 分析範囲の定義**
```
確認項目:
- 分析期間（例: 過去6ヶ月、1年）
- 対象商談の基準（金額、業界、競合等）
- Win/Lossの定義（成約/失注の基準）
- 分析の深度（定量のみ/インタビュー含む）
```

**2. データソースの特定**
- CRM/SFAデータ
- 営業担当者からの情報
- 顧客インタビュー（可能な場合）
- 競合情報

**3. 分析チームの構成**
- 営業マネージャー
- マーケティング担当
- 製品担当
- 分析担当

### Phase 2: データ収集

**1. 定量データの収集**

| データ項目 | 内容 |
|----------|------|
| 商談情報 | 金額、期間、ステージ |
| 顧客情報 | 業界、規模、地域 |
| 競合情報 | 競合名、競合価格 |
| 結果情報 | Win/Loss、理由コード |

**2. 定性データの収集**

営業担当者インタビュー質問例:
```
- この商談の経緯を教えてください
- 勝った（負けた）決め手は何でしたか？
- 競合はどのようなアプローチでしたか？
- 顧客の最終的な決定基準は何でしたか？
- 振り返って改善できる点はありますか？
```

顧客インタビュー質問例:
```
- 今回の選定で重視した基準は何でしたか？
- 各社の提案で印象に残った点は？
- 最終決定に至った理由を教えてください
- 当社への改善要望はありますか？
```

### Phase 3: データ分析

**1. 勝敗の分類**

| カテゴリ | 勝因例 | 敗因例 |
|---------|--------|--------|
| 製品 | 機能優位、統合性 | 機能不足、使いにくさ |
| 価格 | 価格競争力、TCO | 高価格、隠れコスト |
| 営業 | 提案力、関係構築 | 対応遅延、提案品質 |
| ブランド | 信頼性、実績 | 認知度不足 |
| サポート | サポート体制、SLA | サポート不安 |
| その他 | タイミング、政治的要因 | 予算凍結、組織変更 |

**2. パターン分析**

```
分析軸:
- 競合別の勝率・敗因
- 業界別の勝率・敗因
- 金額帯別の勝率・敗因
- 営業担当者別の勝率・敗因
- 商談期間と勝率の相関
```

**3. 統計分析**

| 指標 | 計算式 | 活用 |
|-----|--------|------|
| 勝率 | Win数 / (Win + Loss) | 全体パフォーマンス |
| 競合別勝率 | 対競合A Win / 対競合A 全体 | 競合対策の優先度 |
| 敗因頻度 | 特定敗因 / 全敗因 | 改善の優先順位 |

### Phase 4: インサイト導出

**1. 根本原因分析**

5 Whys手法の活用:
```
例: 価格で負けた
Why 1: なぜ価格で負けた？→ 競合より20%高かった
Why 2: なぜ20%高かった？→ 値引きが限られていた
Why 3: なぜ値引きできなかった？→ 承認プロセスに時間がかかった
Why 4: なぜ時間がかかった？→ 複数階層の承認が必要だった
Why 5: なぜ複数承認？→ 値引きポリシーが硬直的だった
```

**2. 競合別インサイト**

| 競合 | 彼らの強み | 彼らの弱み | 当社の対策 |
|-----|----------|----------|----------|
| A社 | 価格競争力 | サポート弱い | TCO訴求 |
| B社 | 機能豊富 | 複雑で使いにくい | 使いやすさ訴求 |
| C社 | ブランド力 | 柔軟性低い | カスタマイズ性訴求 |

### Phase 5: アクション策定

**1. 短期アクション（1-3ヶ月）**
- 営業トレーニングの実施
- 提案資料の改善
- 価格戦略の調整
- 競合対策トークの整備

**2. 中期アクション（3-6ヶ月）**
- 製品機能の改善
- サポート体制の強化
- パートナーシップの構築

**3. 長期アクション（6ヶ月以上）**
- 製品ロードマップへの反映
- ブランド施策の展開
- 市場ポジショニングの見直し

## インタビュー設計

### 営業担当者向けテンプレート

```markdown
## Win/Loss分析 営業インタビューシート

**商談情報**
- 顧客名:
- 商談金額:
- 競合:
- 結果: Win / Loss
- 商談期間:

**インタビュー質問**

1. 商談の背景
   - 顧客の課題・ニーズは何でしたか？
   - なぜ今回検討を始めたのですか？

2. 競合状況
   - どの競合と競争しましたか？
   - 競合の提案内容は把握していますか？

3. 評価プロセス
   - 顧客の評価基準は何でしたか？
   - 意思決定者は誰でしたか？

4. 勝敗要因
   - 勝った（負けた）最大の理由は何ですか？
   - 他に影響した要因はありますか？

5. 学び
   - 振り返って改善できる点は？
   - 他の商談で活かせる学びは？
```

### 顧客向けテンプレート

```markdown
## Win/Loss分析 顧客インタビューガイド

**導入（2分）**
- 本日は貴重なお時間をいただきありがとうございます
- 今後のサービス改善のため、率直なご意見をお聞かせください
- 約15-20分を予定しています

**質問セクション（15分）**

1. 選定プロセスについて
   - 今回の選定で最も重視した基準は何でしたか？
   - 評価プロセスはどのように進められましたか？

2. 各社の評価について
   - 当社の提案で評価いただいた点は？
   - 当社の提案で不足していた点は？
   - 選定された企業の決め手は？

3. 改善へのフィードバック
   - 当社が改善すべき点を挙げるとすれば？
   - 今後期待することは？

**クロージング（2分）**
- 貴重なご意見ありがとうございました
- いただいたフィードバックは必ず改善に活かします
```

## 分析レポートの構成

### 推奨構成

1. **エグゼクティブサマリー**
   - 分析期間と対象
   - 主要な発見事項（3-5点）
   - 推奨アクション（優先順位付き）

2. **定量分析結果**
   - 全体勝率の推移
   - 競合別勝率
   - 敗因カテゴリ別の分布

3. **定性分析結果**
   - 主要な勝因パターン
   - 主要な敗因パターン
   - 競合別の傾向

4. **競合別分析**
   - 競合Aに対する勝敗傾向
   - 競合Bに対する勝敗傾向
   - （必要に応じて追加）

5. **推奨アクション**
   - 短期アクション
   - 中長期アクション
   - 担当と期限

6. **次回分析への提言**
   - データ収集の改善点
   - 分析手法の改善点

---

## ベストプラクティス

1. **定期的な実施**: 四半期ごと、または商談数に応じて実施
2. **バイアス排除**: 営業担当者の主観だけでなく、顧客の声を収集
3. **迅速なフィードバック**: 分析結果を速やかに関係者と共有
4. **アクション追跡**: 策定したアクションの実行状況を追跡
5. **継続的改善**: 分析プロセス自体も定期的に見直し
6. **経営層への報告**: 重要なインサイトは経営層に共有

---

## よくある失敗パターン

| 失敗パターン | 原因 | 対策 |
|------------|------|------|
| 営業のバイアス | 自己正当化の傾向 | 顧客インタビューを追加 |
| サンプル不足 | 分析対象が少なすぎる | 分析期間を延長 |
| アクションなし | 分析で終わってしまう | アクション策定を必須化 |
| 一回限り | 継続しない | 定期実施をプロセス化 |
| 表面的分析 | 根本原因に至らない | 5 Whysで深掘り |
