# Competitive Analysis Frameworks

競合分析で使用する主要フレームワークのリファレンスガイド。

---

## 1. Porter's Five Forces（5つの競争要因）

業界の競争環境と収益性を分析するフレームワーク。

### 5つの要因

| 要因 | 分析観点 | 評価項目 |
|-----|---------|---------|
| **業界内の競争** | 既存競合との競争激しさ | 競合数、成長率、差別化度、固定費比率 |
| **新規参入の脅威** | 新規プレイヤーの参入障壁 | 規模の経済、ブランド、資本要件、規制 |
| **代替品の脅威** | 代替製品・サービスの存在 | 代替品の価格性能比、スイッチングコスト |
| **買い手の交渉力** | 顧客の価格交渉力 | 購入量、情報量、スイッチングコスト |
| **供給者の交渉力** | サプライヤーの影響力 | 供給者の集中度、代替可能性 |

### 活用シーン
- 新市場参入の判断
- 業界の魅力度評価
- 戦略的ポジショニングの検討

---

## 2. SWOT分析

内部要因（強み・弱み）と外部要因（機会・脅威）を整理するフレームワーク。

### マトリクス構成

|  | ポジティブ | ネガティブ |
|--|----------|----------|
| **内部要因** | Strengths（強み） | Weaknesses（弱み） |
| **外部要因** | Opportunities（機会） | Threats（脅威） |

### 分析のポイント

**Strengths（強み）**
- 競合より優れている点
- 顧客が評価するポイント
- 独自のリソース・能力

**Weaknesses（弱み）**
- 競合に劣る点
- 改善が必要な領域
- リソースの不足

**Opportunities（機会）**
- 市場トレンドの追い風
- 競合の弱点を突ける領域
- 新技術・規制変化

**Threats（脅威）**
- 競合の動き
- 市場環境の変化
- 技術の陳腐化

### TOWS分析への発展

| | Strengths | Weaknesses |
|--|----------|-----------|
| **Opportunities** | SO戦略: 強みで機会を活かす | WO戦略: 弱みを克服し機会を活かす |
| **Threats** | ST戦略: 強みで脅威に対抗 | WT戦略: 弱みと脅威を最小化 |

## 3. 競合ポジショニングマップ

2軸で競合各社の位置づけを可視化するフレームワーク。

### 代表的な軸の組み合わせ

| 軸1 | 軸2 | 用途 |
|-----|-----|------|
| 価格（高-低） | 品質（高-低） | 価値訴求の比較 |
| 機能（多-少） | 使いやすさ（高-低） | 製品特性の比較 |
| 市場シェア（大-小） | 成長率（高-低） | 市場ポジションの比較 |
| カスタマイズ性（高-低） | 導入スピード（速-遅） | 提供方式の比較 |

### 作成手順
1. 分析目的に合った2軸を選定
2. 各軸の評価基準を定義
3. 自社と競合各社をプロット
4. クラスター（群）を特定
5. 空白地帯（ホワイトスペース）を発見

### 活用ポイント
- 差別化ポイントの視覚化
- 新規参入機会の発見
- ポジショニング戦略の検討

---

## 4. 4P/4C比較分析

マーケティングミックスの観点で競合を比較するフレームワーク。

### 4P（企業視点）と4C（顧客視点）の対応

| 4P（企業視点） | 4C（顧客視点） | 比較項目 |
|---------------|---------------|---------|
| **Product**（製品） | Customer Value（顧客価値） | 機能、品質、デザイン、ブランド |
| **Price**（価格） | Cost（コスト） | 価格、TCO、支払条件 |
| **Place**（流通） | Convenience（利便性） | 購入チャネル、納期、サポート体制 |
| **Promotion**（販促） | Communication（コミュニケーション） | 広告、営業活動、PR |

### 比較表テンプレート

| 項目 | 自社 | 競合A | 競合B | 競合C |
|-----|------|-------|-------|-------|
| 製品機能 | | | | |
| 価格帯 | | | | |
| 販売チャネル | | | | |
| マーケティング | | | | |

## 5. Blue Ocean Strategy（戦略キャンバス）

競争を避け、新たな市場空間を創造するためのフレームワーク。

### 戦略キャンバスの作成

**横軸**: 業界で競争している要素（機能、価格、サービス等）
**縦軸**: 各要素への投資レベル（高-低）

### 4つのアクション

| アクション | 説明 | 例 |
|----------|------|-----|
| **Eliminate（排除）** | 業界で当然とされているが不要な要素 | 過剰機能、無駄なオプション |
| **Reduce（削減）** | 業界標準より大幅に削減すべき要素 | 高コスト機能、複雑なUI |
| **Raise（増加）** | 業界標準より大幅に増加すべき要素 | カスタマイズ性、サポート品質 |
| **Create（創造）** | 業界がまだ提供していない要素 | 新しい価値提案、革新的機能 |

### 活用シーン
- 差別化戦略の立案
- 新市場創造の検討
- 競争回避戦略の策定

## 6. Value Curve分析

競合との差別化ポイントを可視化するフレームワーク。

### 作成手順

1. **競争要素の特定**
   - 業界で重視される要素をリストアップ
   - 例: 価格、機能、品質、サポート、ブランド

2. **評価スケールの設定**
   - 各要素を1-10で評価
   - 客観的な基準を設定

3. **自社と競合のプロット**
   - 各要素のスコアをプロット
   - 折れ線グラフで可視化

4. **差異の分析**
   - 自社が優位な領域
   - 競合が優位な領域
   - 差別化の機会

### 分析テンプレート

| 競争要素 | 自社 | 競合A | 競合B | 業界平均 |
|---------|------|-------|-------|---------|
| 価格競争力 | 7 | 5 | 8 | 6 |
| 機能充実度 | 8 | 9 | 6 | 7 |
| 使いやすさ | 9 | 6 | 7 | 7 |
| サポート品質 | 8 | 7 | 5 | 6 |
| ブランド力 | 6 | 8 | 4 | 6 |

## 7. 競争優位性の分析

持続可能な競争優位の源泉を特定するフレームワーク。

### VRIOフレームワーク

| 基準 | 質問 | 意味 |
|-----|------|------|
| **Value**（価値） | その資源は価値を生むか？ | 顧客価値の創出 |
| **Rarity**（希少性） | その資源は希少か？ | 競合が持っていない |
| **Imitability**（模倣困難性） | 模倣は困難か？ | コピーが難しい |
| **Organization**（組織） | 活用する組織能力があるか？ | 実行力の有無 |

### 競争優位の持続性評価

| V | R | I | O | 競争上の位置づけ |
|---|---|---|---|---------------|
| No | - | - | - | 競争劣位 |
| Yes | No | - | - | 競争均衡 |
| Yes | Yes | No | - | 一時的優位 |
| Yes | Yes | Yes | No | 未活用の優位 |
| Yes | Yes | Yes | Yes | 持続的優位 |

---

## フレームワーク選択ガイド

| 分析目的 | 推奨フレームワーク |
|---------|------------------|
| 業界全体の構造理解 | Porter's Five Forces |
| 個別競合との比較 | SWOT、4P/4C比較 |
| 市場でのポジション把握 | ポジショニングマップ |
| 差別化戦略の検討 | Value Curve、Blue Ocean |
| 競争優位の源泉特定 | VRIO分析 |
| 戦略オプションの導出 | TOWS分析 |

---

## ベストプラクティス

1. **複数フレームワークの組み合わせ**: 一つのフレームワークに頼らず、複数の視点で分析
2. **定期的な更新**: 市場環境の変化に応じて分析を更新
3. **データに基づく評価**: 主観的判断を避け、客観的データを使用
4. **アクションへの繋げ**: 分析結果を具体的な戦略アクションに落とし込む
5. **チームでの議論**: 多様な視点を取り入れるためチームで分析・議論
