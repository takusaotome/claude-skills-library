## 🎨 視覚デザイン

### 📱 フォント・レイアウト
- [ ] **可読性の確保**（見出し36-60pt、本文24-30pt、行間120%）
- [ ] **フォント統一**（最大2種類まで、サンセリフ推奨）
- [ ] **ビジュアルヒエラルキー**（重要度に応じた要素の大きさ・位置）
- [ ] **グリッド揃え・均等マージン**（20-40px推奨）
- [ ] **余白の効果的活用**（要素サイズの1.5倍を目安）
- [ ] **フッタークリアランス確保**（最後の要素から底部まで100px以上）
- [ ] **コンテンツ密度調整**（テーブル最大5行、リスト最大8項目）
- [ ] **表紙タイトル長さ別調整**（長いタイトルは小さいフォント・改行を検討）
- [ ] **カスタムフォントサイズ適用**（必要に応じてインラインCSS使用）

### 🎨 色彩・画像
- [ ] **カラーパレット統一**（主要3色+強調1色+警告1色）
- [ ] **高コントラスト比**（WCAG 2.2準拠：4.5:1以上）
- [ ] **色だけに依存しない情報伝達**（形状・パターン併用）
- [ ] **高品質な画像使用**（4K対応・写実的な画像を優先）
- [ ] **ブランドカラーの一貫性**
- [ ] **色の意味統一**（緑=成功、オレンジ=注意、青=情報、赤=エラー）
- [ ] **カラフルボックスの効果的使用**（情報分類に応じた色分け）

### 📦 視覚デザイン要素
- [ ] **CSS box-shadow 不使用**（PDF出力時に灰色四角形になるため、`border` で代替すること）
- [ ] **情報ボックスの適切な使用**（.info-box, .success-box, .warning-box, .error-box）
- [ ] **メトリック表示の明確性**（.metric-box, .metric-number使用）
- [ ] **ステップ表示の視認性**（.step-card, .step-number使用）
- [ ] **タイムライン表示**（.timeline-item, .timeline-badge使用）
- [ ] **アイコンとテキストの組み合わせ**（.icon-text パターン使用）
- [ ] **バッジシステム活用**（.badge, .success-badge, .warning-badge）
- [ ] **グリッドレイアウト**（.grid-2, .grid-3, .grid-4の適切な使用）
- [ ] **ハイライト効果**（.highlight クラスで重要語句を強調）
- [ ] **ホワイトスペース活用**（適切な余白で読みやすさ向上）
