## 5. HMW評価

### 粒度チェック

| HMW | 広すぎ？ | 狭すぎ？ | ちょうど良い |
|-----|---------|---------|-------------|
| #1 | ☐ | ☐ | ☐ |
| #2 | ☐ | ☐ | ☐ |
| #3 | ☐ | ☐ | ☐ |
| #4 | ☐ | ☐ | ☐ |
| #5 | ☐ | ☐ | ☐ |

**粒度の目安:**
- 広すぎ: 1つの質問から10以上のアイデアが出そう
- 狭すぎ: 答えが1-2個しか思いつかない
- ちょうど良い: 5-10個のアイデアが出せそう
