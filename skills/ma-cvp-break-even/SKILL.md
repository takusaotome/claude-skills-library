# CVP / Break-Even Analysis

## Overview

Cost-Volume-Profit analysis determines the relationship between costs, volume, and profit to find the break-even point and plan for target profits.

## Key Formulas

### Break-Even Point
- **In sales**: BEP = Fixed Costs / Contribution Margin Ratio
- **In units**: BEP = Fixed Costs / Unit Contribution Margin

### Contribution Margin
- **Total CM**: Sales - Variable Costs
- **CM Ratio**: CM / Sales
- **Unit CM**: Selling Price per Unit - Variable Cost per Unit

### Margin of Safety
- **Amount**: Current Sales - Break-Even Sales
- **Ratio**: (Current Sales - BEP) / Current Sales * 100%

### Target Profit Sales
- Required Sales = (Fixed Costs + Target Profit) / CM Ratio

## Analysis Workflow

1. **Identify cost structure**: Fixed vs. variable costs
2. **Calculate CM ratio**: From sales and variable cost data
3. **Compute BEP**: Using the appropriate formula
4. **Assess margin of safety**: How far from break-even?
5. **Scenario analysis**: What-if with different assumptions

## Multi-Product Analysis

For businesses with multiple products:
1. Calculate weighted average CM ratio using sales mix
2. Use weighted CM ratio for BEP calculation
3. Note that BEP assumes constant sales mix

## Limitations and Assumptions

- Linear cost behavior within relevant range
- Constant sales mix for multi-product analysis
- Single-period analysis (no time value of money)
- Selling price remains constant
