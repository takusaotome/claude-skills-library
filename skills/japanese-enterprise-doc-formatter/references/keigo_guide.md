# Keigo (敬語) Usage Guide for Enterprise Documents

## Overview

敬語 (Keigo) is the honorific language system in Japanese that shows respect and maintains proper social hierarchy. Proper keigo usage is essential for Japanese enterprise documents.

---

## Three Types of Keigo

### 1. 尊敬語 (Sonkeigo) - Respectful Language

Used to elevate the actions, state, or things of the listener or a third party being respected.

**When to Use:**
- Describing actions of superiors (役員, 部長, etc.)
- Referring to decisions made by management
- Quoting statements from executives

**Common Transformations:**

| Plain Form | 尊敬語 | Usage Example |
|------------|--------|---------------|
| いる | いらっしゃる | 社長はいらっしゃいますか |
| 行く | いらっしゃる / おいでになる | 会議にいらっしゃいますか |
| 来る | いらっしゃる / おいでになる | お客様がいらっしゃいました |
| 言う | おっしゃる | 部長がおっしゃった通り |
| 見る | ご覧になる | 資料をご覧ください |
| 知る | ご存知である | ご存知の通り |
| 食べる | 召し上がる | お召し上がりください |
| する | なさる | どうなさいますか |
| くれる | くださる | ご指導くださいまして |
| 聞く | お聞きになる | お聞きになりましたか |
| 読む | お読みになる | 報告書をお読みになる |
| 考える | お考えになる | どうお考えでしょうか |

**Prefix Patterns:**

| Pattern | Example |
|---------|---------|
| お + verb stem + になる | お読みになる, お考えになる |
| お + verb stem + ください | お読みください, お考えください |
| ご + noun + になる | ご検討になる, ご承認になる |
| ご + noun + ください | ご確認ください, ご検討ください |

### 2. 謙譲語 (Kenjougo) - Humble Language

Used to lower one's own actions to show respect to the listener.

**When to Use:**
- Describing your own actions to superiors
- Describing your department's actions to other departments
- Making requests to management

**Common Transformations:**

| Plain Form | 謙譲語 | Usage Example |
|------------|--------|---------------|
| いる | おる | 担当しております |
| 行く | 参る / 伺う | 明日伺います |
| 来る | 参る | 会議に参ります |
| 言う | 申す / 申し上げる | 以下の通り申し上げます |
| 見る | 拝見する | 資料を拝見いたしました |
| 知る | 存じる / 存じ上げる | 存じております |
| 聞く | 伺う / 拝聴する | ご意見を伺いたく |
| 会う | お目にかかる | お目にかかれて光栄です |
| もらう | いただく | ご承認いただく |
| する | いたす / させていただく | 報告いたします |
| あげる | 差し上げる | 資料を差し上げます |
| 思う | 存じる | ～と存じます |

**Prefix Patterns:**

| Pattern | Example |
|---------|---------|
| お + verb stem + する | お送りする, お届けする |
| お + verb stem + いたす | お送りいたします |
| ご + noun + する | ご報告する, ご連絡する |
| ご + noun + いたす | ご報告いたします |

### 3. 丁寧語 (Teineigo) - Polite Language

General politeness markers that don't elevate or humble anyone specifically.

**When to Use:**
- General business communication
- Written documents (baseline politeness)
- Communication with peers

**Common Transformations:**

| Plain Form | 丁寧語 |
|------------|--------|
| だ | です |
| である | でございます |
| する | します |
| ある | あります |
| いる | います |
| 〜ない | 〜ません |
| 〜た | 〜ました |

## Keigo Levels by Document Type and Audience

### Level Matrix

| Audience | Document Type | Keigo Level | Example Ending |
|----------|---------------|-------------|----------------|
| 役員 | 稟議書 | 最上級 | ～させていただきたく存じます |
| 役員 | 報告書 | 最上級 | ～でございます |
| 本部長 | 稟議書 | 上級 | ～いただきたく存じます |
| 部長 | 稟議書 | 上級 | ～いただきたく存じます |
| 部長 | 提案書 | 標準 | ～いたします |
| 部長 | 購入申請 | 標準 | ～をお願いいたします |
| 課長 | 依頼書 | 標準 | ～をお願いいたします |
| 同僚 | 連絡書 | 基本 | ～します |

### Level Definitions

#### 最上級敬語 (Highest Formal)

**Target**: 役員, 取締役会
**Characteristics**: Double honorifics, maximum humility

**Patterns:**
```
～させていただきたく、ご高配を賜りたく存じます
～についてご裁可を賜りたく、お願い申し上げます
何卒ご承認賜りますよう、お願い申し上げます
ご多忙中誠に恐縮ではございますが～
```

**Example Paragraph:**
```
本件につきまして、以下の通りご承認を賜りたく、謹んでお願い申し上げます。
ご多忙のところ誠に恐縮ではございますが、
ご高覧の上、何卒ご裁可賜りますようお願い申し上げます。
```

#### 上級敬語 (Upper Formal)

**Target**: 本部長, 部長
**Characteristics**: Standard honorifics with humble forms

**Patterns:**
```
～いただきたく存じます
～についてご承認をお願いいたします
ご検討いただければ幸いでございます
```

**Example Paragraph:**
```
本件につきまして、ご承認いただきたく、お願い申し上げます。
ご多忙のところ恐れ入りますが、
ご検討のほど、よろしくお願いいたします。
```

#### 標準敬語 (Standard Formal)

**Target**: 課長, 他部署
**Characteristics**: Basic honorifics and polite forms

**Patterns:**
```
～いたします
～をお願いいたします
ご確認をお願いいたします
```

**Example Paragraph:**
```
本件についてご報告いたします。
ご不明な点がございましたら、お問い合わせください。
よろしくお願いいたします。
```

#### 基本丁寧語 (Basic Polite)

**Target**: 同僚, 部内連絡
**Characteristics**: です/ます form

**Patterns:**
```
～します
～です
～ください
```

## Common Phrases for Enterprise Documents

| Situation | Phrase |
|-----------|--------|
| 新規提案 | このたび、～について提案させていただきます |

| 依頼 | ～につきまして、お願い申し上げます |

|-------|--------|

| 標準 | ご対応をお願いいたします |
| 基本 | 対応をお願いします |

## Keigo Transformation Rules

### Rule 1: Consistent Level Throughout

Maintain the same keigo level throughout the document. Do not mix levels.

**Wrong:**
```
本件についてご承認を賜りたく存じます。(最上級)
詳細は以下の通りです。(基本)
```

**Correct:**
```
本件についてご承認を賜りたく存じます。(最上級)
詳細は以下の通りでございます。(最上級)
```

### Rule 2: Subject Determines Form

| Subject | Use |
|---------|-----|
| Superior's action | 尊敬語 |
| Own/company action | 謙譲語 |
| Third party/neutral | 丁寧語 |

### Rule 3: Double Honorific Avoidance

Avoid stacking multiple honorific forms (二重敬語).

**Wrong:**
- お読みになられる (お～になる + られる)
- ご覧になられる

**Correct:**
- お読みになる
- ご覧になる

**Exception**: Some double forms are accepted through common usage:
- お召し上がりになる (accepted)
- お見えになる (accepted)

### Rule 4: Direction-Based Selection

| Action Direction | Form |
|------------------|------|
| Superior → You | 尊敬語 (くださる) |
| You → Superior | 謙譲語 (いただく) |
| You ← Superior (receive) | 尊敬語 + 謙譲語 |

## Common Errors to Avoid

### Error 1: Incorrect Verb Forms

| Incorrect | Correct |
|-----------|---------|
| 拝見させていただきます | 拝見いたします |
| お伺いさせていただきます | 伺います |
| ご連絡させていただきます | ご連絡いたします |

### Error 2: Wrong Subject-Form Pairing

| Incorrect | Correct |
|-----------|---------|
| 部長が申しました | 部長がおっしゃいました |
| 私がおっしゃった | 私が申し上げた |
| お客様が参ります | お客様がいらっしゃいます |

### Error 3: Overuse of させていただく

「させていただく」should only be used when:
1. You received permission from someone
2. You benefit from the action

**Wrong:**
```
資料を送らせていただきます（許可不要の場合）
```

**Correct:**
```
資料をお送りいたします
```

### Error 4: Mixing です/である Forms

| Incorrect | Correct |
|-----------|---------|
| 理由は以下の通りである。効果は大きいです。 | 理由は以下の通りです。効果は大きいです。 |

## Keigo Quick Reference Card

### For 稟議書 (Ringi) - Highest Level

| Section | Keigo Pattern |
|---------|---------------|
| 件名 | ～の件 / ～について |
| 目的 | ～についてご承認を賜りたく存じます |
| 背景 | ～でございます |
| 効果 | ～と考えております |
| 結び | 何卒ご裁可賜りますようお願い申し上げます |

### For 購入申請 (Purchase Request) - Standard Level

| Section | Keigo Pattern |
|---------|---------------|
| 件名 | ～の購入について |
| 理由 | ～が必要でございます |
| 依頼 | ご承認をお願いいたします |
| 結び | よろしくお願いいたします |

### For 報告書 (Report) - Standard Level

| Section | Keigo Pattern |
|---------|---------------|
| 件名 | ～に関する報告 |
| 概要 | ～について報告いたします |
| 結果 | ～でございました |
| 結び | 以上、ご報告いたします |
